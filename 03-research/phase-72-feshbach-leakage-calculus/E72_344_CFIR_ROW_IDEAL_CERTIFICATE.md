# E72.344 -- CFIR row-ideal certificate

**Purpose.** Convert `FINITE-CFIR` from E72.342--E72.343 into a finite row-ideal identity.  The
important observation is that `Def_CFIR,T` is linear in the pole-even ground vector `xi`.  Therefore it
should be proved by the eigenvector equation `(C_E-mu I)xi=0`, not by estimating the entries of `xi`.

## 1. The finite CFIR functional is linear in `xi`

After E72.343, every term in

```text
Def_CFIR,T
= J_T[
   T_W02(z)[xi] - T_WR(z)[xi] - T_Prime(z)[xi]
   - 2kappa_*G_x(z)
   - I_T^H G_x(z)
   + Tail_T^M(z)
]
```

is a finite linear functional of `xi`.  Hence there is an explicit finite matrix

```text
R_T(L)
```

whose rows are indexed by Hermite jet slots `(a_j,p)` and whose columns are active mesh indices `n`,
such that

```text
Def_CFIR,T = R_T(L) xi.                               (1)
```

The entries of `R_T(L)` are obtained by differentiating, in `z`, the explicit finite kernels:

```text
KQ_z^M(y;n),        G basis  (1-e^(zL))/(iz-d_n),
I_T^H basis,        TailBasis_{z,n}^M(a_j).
```

Thus `R_T(L)` is finite and directly computable from the same data as `C_E`.

## 2. Row ideal generated by the eigenvector equation

Let

```text
E_L:=C_E-mu I
```

on the pole-even active mesh.  The ground vector satisfies

```text
E_L xi=0.                                             (2)
```

For any finite matrix `A_T(L)` with the same number of columns as `R_T`,

```text
A_T(L)E_L xi=0.
```

Therefore, if one can find `A_T(L)` and a residual row matrix `S_T(L)` such that

```text
R_T(L)=A_T(L)E_L+S_T(L),                              (3)
```

then

```text
Def_CFIR,T=S_T(L)xi.                                  (4)
```

Consequently `FINITE-CFIR` follows from

```text
||S_T(L)|| <= C_TL^B
```

in any fixed norm where `||xi||=1`.

The exact certificate is the special case

```text
R_T(L) in Row(E_L),       i.e. S_T(L)=0.              (5)
```

Then `Def_CFIR,T=0` identically for every eigenvector `xi`.

## 3. Orthogonal-projector form

Let `P_null` denote the orthogonal projection onto `ker(E_L)` in the finite pole-even mesh.  Since the
relevant eigenspace is one-dimensional in the pole-even ground channel,

```text
P_null = xi xi^*
```

after normalization.  The row-ideal condition is equivalent to

```text
R_T(L)P_null = 0.                                     (6)
```

For a polynomial bound instead of exact vanishing, it is enough to prove

```text
||R_T(L)P_null|| <= C_TL^B.                           (7)
```

This is still a finite statement, but (3) is the non-circular form because it uses `E_L` directly and
does not name `xi` except through the eigenvector equation.

## 4. Bordered-minor test

A row `r` lies in `Row(E_L)` if and only if every maximal minor of the augmented matrix

```text
[ E_L ]
[ r   ]
```

that replaces a row of `E_L` by `r` vanishes, with the usual rank convention for the one-dimensional
kernel.  For the full CFIR matrix `R_T`, exact row-ideal membership is the finite collection of
determinantal identities

```text
det ReplaceRow(E_L; i, R_T[(a_j,p),*]) = 0            (8)
```

for every Hermite row `(a_j,p)` and every admissible replacement row `i`.

Equivalently, in adjugate form,

```text
R_T(L) adj(E_L)=0,                                    (9)
```

where `adj(E_L)` is interpreted at the simple ground eigenvalue by the reduced adjugate on the
pole-even complement.

This is a finite algebraic identity.  It is the CFIR analogue of the bordered determinant endpoint
E72.97, but now attached to the transformed compact-root window.

## 5. Relation to the Cauchy block

If exact membership (5) holds, then `FINITE-CFIR` is immediate and the E72.324--E72.325 Cauchy block
forces off-line nodal suppression.

If only the polynomial residual version (3) holds, the same conclusion follows because the maximal
off-line block has inverse of size `e^{-alpha L}L^B` on every fixed cluster:

```text
H_T(L)J_TG_x = -J_TTail_T^M + O(L^B)
```

and the leading `-diag(e^(a_jL))C_conf` is invertible.

Thus the row-ideal certificate is exactly strong enough for the transformed route.

## 6. Current endpoint

The Phase 72 analytic endpoint is now:

```text
CFIR-ROW:
Find finite matrices A_T(L), S_T(L) such that
R_T(L)=A_T(L)(C_E-mu I)+S_T(L),
with ||S_T(L)||<=C_TL^B.
```

Exact closure is the sharper identity `S_T=0`, equivalently the determinant/adjugate identities
(8)--(9).

All objects in `CFIR-ROW` are finite:

```text
C_E, mu, R_T, A_T, S_T,
```

and `R_T` has explicit entries from E72.342--E72.343.
