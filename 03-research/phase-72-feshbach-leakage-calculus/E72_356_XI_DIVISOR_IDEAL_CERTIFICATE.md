# E72.356 - Xi-divisor ideal certificate for CFIR compatibility

## 1. Purpose

E72.355 proves that a node-blind CFIR closure is false.  Therefore the missing identity must use the
fact that the window nodes are zeros of `Xi`, but it must do so without assuming RH and without
replacing Xi-zero nodes by finite Weyl roots.

This note gives the precise algebraic form of such an identity: the Lambda-elimination minors must
belong to the divisor ideal generated by the completed function.

## 2. One-row analytic families

For a simple node variable `z`, define the finite row families

```text
K_z := k_z,
S_z := - sum_{b in W_T} K_L(z,b) k_b
       + sum_{b in W_T} TailBasis_z^M(b),
```

or, in the exact version, the corresponding rows obtained from the full `S_T^{Xi}` construction.

Apply the reduced adjugate:

```text
Y(z) := K_z Adj(E),
X(z) := S_z Adj(E).                                  (1)
```

For a simple ground channel, `Adj(E)=gamma xi xi^*`, so `X(z),Y(z)` are rank-one row multiples.  The
following formulation does not require choosing `xi`.

## 3. Compatibility minors as two-variable functions

For columns `c,d` of `Adj(E)`, define the two-variable minor

```text
Delta_{c,d}(z,w)
:= X(z)_c Y(w)_d - X(w)_d Y(z)_c.                    (2)
```

In the rank-one channel, this is the adjugate form of

```text
(S_z xi)(K_w xi) - (S_w xi)(K_z xi).
```

By E72.354, scalar Lambda compatibility on a zero window is equivalent to the vanishing of all such
minors at every pair of window nodes:

```text
Delta_{c,d}(a_j,a_k)=0.                              (3)
```

## 4. Divisor-ideal sufficient condition

Let

```text
Z(z)=Xi(1/2+z)
```

in shifted variables.  A non-circular sufficient condition for (3) is the ideal membership

```text
Delta_{c,d}(z,w)
= A_{c,d}(z,w) Z(z) + B_{c,d}(z,w) Z(w)              (4)
```

with `A_{c,d},B_{c,d}` holomorphic on the fixed compact window.

Indeed, if `Z(a_j)=Z(a_k)=0`, then (4) gives

```text
Delta_{c,d}(a_j,a_k)=0.
```

For multiplicities, the Hermite version is membership in the corresponding jet ideal:

```text
Delta_{c,d}(z,w) in ( Z(z), Z(w) )
```

as a local analytic ideal.  This means every mixed derivative required by the window multiplicities
also vanishes.

## 5. Polynomial version

The transformed route only needs polynomial residual before the exponential Cauchy inversion.  Thus
one may prove the weaker decomposition

```text
Delta_{c,d}(z,w)
= A_{c,d}(z,w) Z(z) + B_{c,d}(z,w) Z(w)
  + Err_{c,d}(z,w;L),                                (5)
```

where, on the compact window,

```text
|Err_{c,d}(z,w;L)| <= C_T L^B
```

in the same normalization as `X,Y`.  Then the Lambda-eliminated minors are polynomial on every
Xi-zero window.

## 6. Why this is the correct replacement for node-blind closure

The shifted controls in E72.354 falsify the identity

```text
Delta_{c,d}(z,w)=0
```

for arbitrary nodes.  E72.356 does not ask for that.  It asks for the weaker and structurally correct
statement:

```text
Delta_{c,d} vanishes on the Xi divisor product.
```

Equivalently, the obstruction is not zero as a function; it is zero modulo the divisor ideal of `Xi`.

This is exactly the mathematical place where the arithmetic divisor enters:

```text
universal kernel identity      false;
Xi-divisor ideal identity      possible and load-bearing.
```

## 7. Finite verification protocol

On a finite rectangle `Omega` avoiding boundary zeros, ideal membership can be tested by Cauchy
division.  Define

```text
Q_{c,d}(z,w)
:= Delta_{c,d}(z,w)/(Z(z)Z(w)).
```

If (4) holds with holomorphic `A,B`, then `Q` has no mixed residue at any pair of zeros:

```text
Res_{z=a_j} Res_{w=a_k} Q_{c,d}(z,w) dz dw = 0.      (6)
```

Conversely, for finite windows with prescribed multiplicities, vanishing of all local mixed
principal parts is equivalent to membership in the local ideal `(Z(z),Z(w))`.

Thus the finite certificate is:

```text
all mixed principal parts of Delta/(Z(z)Z(w)) vanish
on the chosen Xi-zero window.
```

This is a local algebra statement; it does not assume the zeros lie on the critical line.
