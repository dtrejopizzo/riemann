## 0. Scalar endpoint channel

For fixed Cauchy height `s`, define the scalar overlap channel:

```text
a_{x,s}(y) := <v_{x,s}, Q_x(y)k_x>,
v_{x,s}=C_x^(-1)Q_x(sI-D_x)^(-1)1_x.
```

The endpoint-discrepancy scalar is:

```text
D_x(s;z)
 := int_1^x (u/x)^z u^(-1/2) a_{x,s}(log u)dE(u).
```

The remaining target is:

```text
D_x(s;z)->0                                      (SED)
```

on the two Cauchy heights, uniformly for `z` in compact subsets of `Re z>1/2`.

## 1. Endpoint taper in the scalar channel

By the Fourier overlap bound,

```text
|a_{x,s}(y)| <= 2||v_{x,s}||||k_x||(1-y/L).      (A-taper)
```

But this absolute bound is not enough by itself; after integrating against `dE`, PNT leaves a possible
half-power. The proof must use the scalar oscillation of `a_{x,s}(y)`, not merely its size.

## 2. Abel form

Let

```text
F_{x,s,z}(u)=(u/x)^z u^(-1/2)a_{x,s}(log u).
```

Since `a_{x,s}(L)=0`, the endpoint boundary at `u=x` vanishes. Abel summation gives:

```text
D_x(s;z)
 = -int_1^x E(u) partial_u F_{x,s,z}(u)du
   + E(1)F_{x,s,z}(1).
```

Thus `(SED)` follows from:

```text
int_1^x E(u) partial_u F_{x,s,z}(u)du -> 0.      (SED-Abel)
```

## 3. Why this is the exact wall

If one estimates absolutely, PNT gives at best:

```text
o(sqrt(x)) * scalar-channel-size.
```

So absolute PNT closes only with an extra half-power channel bound. Numerics show the uniform channel
is too strong, but the coherent endpoint sum is small.

Therefore the remaining theorem must be:

### Scalar Endpoint Discrepancy Cancellation

For the specific CCM scalar channel `a_{x,s}`,

```text
int_1^x E(u) partial_u[
  (u/x)^z u^(-1/2)a_{x,s}(log u)
]du -> 0.                                      (SEDC)
```

This is weaker than the Phase 70 Hardy-Euler estimate because the test family is no longer arbitrary:
it is the one-dimensional family generated by the compressed Weyl-Feshbach channel.

## 4. Non-circularity gate

`SEDC` is admissible if proved from:

```text
1. the finite CCM eigenvector equation,
2. the prolate complement structure,
3. endpoint overlap/taper,
4. PNT-level input.
```

It is circular if proved by:

```text
RH-scale bounds for E(u),
zero-location resonance cancellation,
or convergence of P_x to Xi.
```
