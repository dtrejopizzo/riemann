# E72.196 - Green-cycle form of the fixed certificate

## Purpose

E72.195 leaves three fixed trace inequalities. This note writes them in the exact arithmetic form in
which they must be proved. The important correction is that the cycles are not bare translation
cycles of `Q_y`; the exact whitening inserts the model Green operator between every prime-power cell.

This is the right object for the next theorem.

## Data

Let `C_x` denote the model complement matrix in the even pole complement, and let

```text
G_x = C_x^(-1).
```

For every prime power `n=p^m<=x`, set

```text
y_n    = log n,
beta_n = Lambda(n)n^(-1/2),
B_n    = - beta_n B_x^* Q_x(y_n) B_x,
```

where `B_x` is the fixed even-complement embedding used in Phase 72. The relative cell used by the
certificate is

```text
A_n = C_x^(-1/2) B_n C_x^(-1/2).
```

The two block kernels are

```text
K_j = sum_{n in S_j(x)} A_n,
S_0(x)={n=p^m<=x : log n <= 0.60L},
S_1(x)={n=p^m<=x : 0.60L < log n <= L}.
```

## Green-cycle identity

For any word `w=(n_1,...,n_r)` of prime powers in a fixed block,

```text
Tr(A_{n_1}...A_{n_r})
 =
(-1)^r (prod_i beta_{n_i})
Tr(G_x B_x^*Q_x(y_{n_1})B_x
   G_x B_x^*Q_x(y_{n_2})B_x
   ...
   G_x B_x^*Q_x(y_{n_r})B_x).                 (GC)
```

This follows by cyclically moving the `C_x^(-1/2)` factors:

```text
Tr(C^{-1/2}B_1C^{-1/2} ... C^{-1/2}B_rC^{-1/2})
 = Tr(G B_1 G B_2 ... G B_r).
```

Thus every trace in the fixed certificate is an explicit finite sum of Green-cycles.

## The three inequalities in Green-cycle form

### NTC-8

For `j=0,1`, with `M_0=0.90`, `M_1=0.60`,

```text
M_j^(-16)
sum_{n_1,...,n_16 in S_j(x)}
(-1)^16 (prod_i beta_{n_i})
Tr(GQ_{n_1}GQ_{n_2}...GQ_{n_16}) < 1.          (GC-NTC)
```

Here `GQ_n` abbreviates `G_x B_x^*Q_x(log n)B_x`.

### LM8-ASRP

Let

```text
P_j(t)=sum_{r=0}^8 c_{j,r}t^r
```

be the certified degree-8 envelope from E72.192. Then

```text
0.90^2 sum_{r=0}^8 c_{0,r} Tr((K_0/0.90)^r)
+0.60^2 sum_{r=0}^8 c_{1,r} Tr((K_1/0.60)^r)
< 0.9409,                                      (GC-LM8)
```

with every `Tr(K_j^r)` expanded by `(GC)`.

### XNEG-32

Let

```text
g_j(t)=sum_{r=0}^{32} d_{j,r}t^r
```

be the degree-32 signed Chebyshev residual polynomial on the certified interval of `K_j`. The mixed
sign gate is

```text
2 Tr(g_0(K_0)g_1(K_1)) <= 0.                    (GC-XNEG32)
```

Equivalently,

```text
2 sum_{r=0}^{32} sum_{s=0}^{32} d_{0,r}d_{1,s}
  sum_{n_1,...,n_r in S_0(x)}
  sum_{m_1,...,m_s in S_1(x)}
  Tr(A_{n_1}...A_{n_r}A_{m_1}...A_{m_s}) <= 0.
```

Each inner trace is again the same Green-cycle `(GC)`, now with a mixed block word.

## Why this is the right target

The earlier bare-cycle viewpoint was incomplete because it hid the exact whitening. The model Green
operator is not a nuisance term: it is the archimedean Feshbach geometry. Any proof that drops it risks
falling back to a scalar whitening or an incoherent absolute-value ceiling.

So the remaining arithmetic theorem is:

> Prove `(GC-NTC)`, `(GC-LM8)`, and `(GC-XNEG32)` uniformly for the exact Green-cycles generated by
> the model complement `C_x` and the zeta prime-power cells.

This is the fixed finite theorem left by Phase 72.

## Next reduction target

The only possible further simplification that preserves the exact whitening is to use the translation
identity

```text
Q_x(y)=U_y+U_y^*
```

inside the Green-cycle. This expands every cycle into signed one-sided translation words with Green
insertions:

```text
Tr(G(U_{eps_1 y_1})G(U_{eps_2 y_2})...G(U_{eps_r y_r})).
```

The proof must show cancellation or domination at this signed Green-word level. Plain Haar/absolute
summation is too coarse, and bare `Q_y` cycles ignore the decisive Feshbach geometry.
