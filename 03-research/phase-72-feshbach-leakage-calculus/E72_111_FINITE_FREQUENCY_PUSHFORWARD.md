## 0. Fourier transform of the reflected discrepancy

Let:

```text
mu_x:=dE_x^leftarrow=dPsi_x^leftarrow-xexp(-r)dr.
```

For real `omega`, define:

```text
Ehat_x(omega) := int_0^L exp(-i omega r)dmu_x(r),
Ehat'_x(omega):= partial_omega Ehat_x(omega).
```

Then for each mesh point `d_n`:

```text
sigma_n^E(tau)
 = int exp(-i tau r)sin(d_nr)dmu_x(r)
 = [Ehat_x(tau-d_n)-Ehat_x(tau+d_n)]/(2i),                  (S)

dot_sigma_n^E(tau)
 = int exp(-i tau r)r cos(d_nr)dmu_x(r)
 = (i/2)[Ehat'_x(tau-d_n)+Ehat'_x(tau+d_n)].                (DS)
```

These formulas are exact and contain the full prime-minus-continuous subtraction.

## 1. Loewner pushforward map

For a vector `k`, define the linear map:

```text
L_k : (sigma,dot_sigma) -> Kk,
```

where `K` is the Loewner matrix generated by `(sigma,dot_sigma)`:

```text
K_mn=(sigma_m-sigma_n)/(d_m-d_n),       m != n,
K_nn=dot_sigma_n.
```

Entrywise:

```text
(L_k(sigma,dot_sigma))_m
 = k_m dot_sigma_m
   + sum_{n != m} k_n (sigma_m-sigma_n)/(d_m-d_n).            (LK)
```

Thus:

```text
K_x^E(tau)k_x = L_{k_x}(sigma^E(tau),dot_sigma^E(tau)).       (PUSH)
```

Combining `(S)`, `(DS)`, and `(LK)`, the post-main source is a finite linear image of the frequency
packet:

```text
Y_x(tau)
 := ( Ehat_x(tau-d_n), Ehat_x(tau+d_n),
      Ehat'_x(tau-d_n), Ehat'_x(tau+d_n) )_{|d_n|<=H'}.
```

No zeta zero is used in this representation.

## 2. Matrix form of MASS and ENERGY

For each fixed physical band `H`, define:

```text
PUSH_{x,H,tau}: Y_x(tau) -> c_{x,H}^{pm}(tau)
```

as the composition:

```text
Y
 -> (sigma,dot_sigma)
 -> K_x^E(tau)k_x
 -> (2/(L sqrt(x)))B_x^*1_{|d|<=H}K_x^E(tau)k_x.
```

Then E72.109 gives:

```text
Mass_x(H,tau)
 = ell_{x,H} PUSH_{x,H,tau}Y_x(tau),
```

and:

```text
Flux_x(H,tau)^2
 = Y_x(tau)^* PUSH_{x,H,tau}^*
   C_E^(-1)B_x^*W_H^*(P_H^0)^*T_H^*T_HP_H^0W_HB_xC_E^(-1)
   PUSH_{x,H,tau}Y_x(tau).                                  (FREQ-Q)
```

This is the finite-frequency version of the double discrepancy energy.

## 3. Why this matters

The old impossible estimate was an incoherent bound on:

```text
sum_n |Ehat_x(tau +/- d_n)| + |Ehat'_x(tau +/- d_n)|.
```

The corrected target is a quadratic form after the full pushforward:

```text
Y_x(tau)^*M_{x,H,tau}Y_x(tau)=O(1),
```

where `M_{x,H,tau}` contains:

```text
Loewner divided differences,
the prolate vector k_x,
post-main scaling 2/(L sqrt(x)),
Feshbach shorting C_E^(-1),
endpoint mass centering,
physical-band cumulative energy.
```

Thus the route no longer requires every Chebyshev Fourier datum to be small.  It requires only that
the finite packet be small in the structured seminorm selected by the Weyl-Feshbach geometry.
