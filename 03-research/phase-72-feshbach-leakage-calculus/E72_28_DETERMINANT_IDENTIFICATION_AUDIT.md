# E72.28 -- Audit of bordered determinant identification

**Date:** 2026-07-08.
**Role:** decide whether the relative bordered Fredholm theorem creates new leverage or simply renames
the Phase 71 convergence problem.

## 0. The issue

E72.27 proves:

```text
trace-norm convergence of relative bordered pencils
+ determinant identification
=> RH.
```

The dangerous clause is:

```text
G(s)det(I+K(s)) = c Xi(s).
```

If this is assumed, RH is immediate but nothing has been proved. It must be derived from finite CCM
data.

## 1. What determinant identification means

Since

```text
P_x(s)=-det B_x(s),
```

the statement

```text
det B_x(s)/det B_x(0) -> Xi(s)/Xi(0)
```

is exactly local uniform convergence of the finite CCM spectral functions.

That is the Phase 71 missing theorem in determinant language.

Thus E72.27 is not itself the proof of the missing convergence. It is a correct closure theorem, but
the determinant identification remains the main work.

## 2. What is genuinely new

The new leverage is not the final identification. It is the replacement of root tracking by a
trace-class current problem:

```text
K_x(s)=(B_x-B_x^0)(B_x^0)^(-1).
```

If `K_x` can be estimated directly from the finite CCM matrix, one can prove convergence of currents
without following individual roots or eigenvectors.

This avoids:

```text
stable clustering,
boundary phase fitting,
Weil positivity,
and scalar Abel modulus arguments.
```

## 3. The exact remaining theorem

The non-circular theorem is:

### Bordered Trace Estimate

Construct `B_x^0` canonically from the prolate model and prove, on every compact `Omega` in the strip,

```text
sup_{s in Omega} ||(B_x-B_x^0)(B_x^0)^(-1)||_1 -> 0.       (BTE)
```

If additionally

```text
det B_x^0(s)/det B_x^0(0) -> Xi(s)/Xi(0)
```

is Connes's prolate model limit in the same gauge, then E72.27 closes RH.

This is stronger than simply saying "determinant identification"; it gives a concrete estimate.

## 4. Non-circularity of the model determinant

The model determinant clause is admissible only if `B_x^0` is built from Connes's prolate vector
`k_lambda` whose Fourier transform converges to `Xi` by the source theorem.

Then the identity

```text
det B_x^0 -> Xi
```

is not an assumption about the actual finite CCM data. It is the known prolate limit.

The actual arithmetic burden becomes exactly `(BTE)`: prove the actual finite bordered pencil is
trace-close to the prolate bordered pencil.

## 5. Relation to Feshbach leakage

`(BTE)` is related to but different from vector leakage:

```text
theta_x -> k_lambda
```

implies bordered trace closeness if the pole mesh and normalizations are controlled. But `(BTE)` may be
weaker because the bordered determinant only sees the current generated by `(D_x,xi_x)`.

So the new route is:

```text
do not prove theta_x -> k_lambda in norm;
prove bordered trace closeness of the Weyl current.
```

## 6. Status

```text
proved: determinant identification without BTE is just Phase 71 convergence;
new:    BTE is the sharp non-circular estimate to attempt;
open:   prove BTE from explicit CCM/prolate data.
```

This is the current endpoint. The next work must be an estimate of the trace norm in `(BTE)`, not
another abstract closure theorem.
