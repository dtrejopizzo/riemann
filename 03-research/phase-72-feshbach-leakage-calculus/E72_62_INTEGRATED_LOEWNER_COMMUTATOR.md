# E72.62 -- Integrated Loewner commutator for the endpoint discrepancy

**Date:** 2026-07-08.
**Role:** convert the reflected LFDC operator into an exact finite commutator equation.

## 0. Reflected discrepancy operator

From E72.61 define the signed endpoint measure:

```text
dE_x^leftarrow(r)=dPsi_x^leftarrow(r)-xexp(-r)dr.
```

For `Re z>1/2`, set:

```text
K_x^E(z)
 := int_0^L exp(-(z-1/2)r)Lo_r dE_x^leftarrow(r).                (K)
```

Then:

```text
<v,[S_x^Lambda(z)-S_x^cont(z)]k>
 = (2/(L sqrt(x)))<v,K_x^E(z)k>.                                  (LFDC-K)
```

## 1. Integrated commutator identity

Let:

```text
Sigma_x^E(z)
 := int_0^L exp(-(z-1/2)r)sin(rD)dE_x^leftarrow(r).
```

### Lemma 72.62.1

The finite matrix `K_x^E(z)` satisfies:

```text
[D,K_x^E(z)] = [Sigma_x^E(z),J].                                  (IKC)
```

### Proof

For each `r`, E72.54 gives:

```text
[D,Lo_r]=[sin(rD),J].
```

The scalar weight `exp(-(z-1/2)r)` commutes with the finite matrices and the Stieltjes integral is
entrywise finite. Integrating gives `(IKC)`. `QED`

Entrywise, for `m != n`:

```text
K_x^E(z)_{mn}
 = [sigma_m(z)-sigma_n(z)]/(d_m-d_n),                              (OFF)
```

where:

```text
sigma_m(z)
 := int_0^L exp(-(z-1/2)r)sin(d_mr)dE_x^leftarrow(r).
```

The diagonal is the endpoint derivative transform:

```text
K_x^E(z)_{nn}
 = int_0^L exp(-(z-1/2)r)r cos(d_nr)dE_x^leftarrow(r)
 = partial_{d_n} sigma_n(z).                                       (DIAG)
```

Thus `K_x^E(z)` is the Loewner matrix generated by the scalar endpoint transforms `sigma_n(z)`.

## 2. Cauchy-Loewner bilinear formula

For any vectors `v,k`:

```text
<v,K_x^E(z)k>
 = sum_{m != n} conj(v_m)k_n [sigma_m-sigma_n]/(d_m-d_n)
   + sum_n conj(v_n)k_n partial_{d_n}sigma_n.                       (BIL)
```

Equivalently, if `H_d` denotes the finite divided-difference Hilbert operator:

```text
<v,K_x^E(z)k>
 = <Hilb_d(v,k),sigma(z)> + <diag(v,k),partial_d sigma(z)>.          (HILB)
```

This is the exact endpoint-reflected version of E72.52, now after subtracting the PNT main term.

## 3. Why incoherent bounds still fail

Classical zero-free PNT gives good bounds for each fixed:

```text
sigma_m(z)
 = int_0^L exp(-(z-1/2)r)sin(d_mr)dE_x^leftarrow(r),
```

but the frequency set grows with the finite CCM window. Summing absolute values in `(BIL)` returns the
same half-power obstruction as E72.51--E72.53.

Therefore the proof cannot estimate `sigma_m` independently. The commutator identity must be paired
with the special vectors:

```text
v=C_x^(-1)Q_x(sI-D)^(-1)1,
k=k_x,
```

and with their equations:

```text
C_xv=Q_x(sI-D)^(-1)1,
H_x^0k=mu_x^0k.
```

## 4. The exact remaining algebraic estimate

Define the coefficient vector:

```text
alpha_n(v,k)
 := sum_{m != n} [conj(v_n)k_m+conj(v_m)k_n]/(d_n-d_m),
```

and the diagonal coefficient:

```text
beta_n(v,k):=conj(v_n)k_n.
```

Then `(BIL)` becomes:

```text
<v,K_x^E(z)k>
 = sum_n alpha_n(v,k)sigma_n(z)
   + sum_n beta_n(v,k)partial_{d_n}sigma_n(z).                      (COF)
```

The LFDC target is therefore equivalent to:

```text
(2/(L sqrt(x)))
| sum_n alpha_n sigma_n + sum_n beta_n partial_{d_n}sigma_n | -> 0. (ECF)
```

## 5. Candidate closure statement

### Endpoint Cauchy-Hilbert Cancellation

For the Feshbach-Cauchy vector `v` and prolate ground vector `k`, uniformly on the two Cauchy heights
and compact `z` windows:

```text
sum_n alpha_n(v,k)sigma_n(z)
 + sum_n beta_n(v,k)partial_{d_n}sigma_n(z)
 = o(L sqrt(x)).                                                    (ECHC)
```

By `(LFDC-K)`, `(ECHC)` implies LFDC. Together with E72.30 and the one-derivative version, it implies
Schur-free current convergence and RH.
