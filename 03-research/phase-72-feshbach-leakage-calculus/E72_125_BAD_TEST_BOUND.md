# E72.125 -- Bad-test square bound

**Date:** 2026-07-09.
**Role:** turn `BUP` into an explicit finite family of scalar endpoint-discrepancy bounds.

## 0. Setup

Let:

```text
Q_x=[e_1,...,e_b]
```

be any orthonormal basis of the bad packet subspace `B_{x,H,q}` of E72.116.  Write each packet vector
as:

```text
e_a=(alpha_{a,n}, beta_{a,n})_n
```

in the `sigma/dot_sigma` coordinates.  Define the scalar bad test:

```text
T_{a,x}(r)
 := sum_n conjugate(alpha_{a,n}) sin(d_nr)
  + sum_n conjugate(beta_{a,n}) r cos(d_nr).                 (BT)
```

Then, for every finite CCM root `tau_j`,

```text
<Y_x(tau_j),e_a>
 = int_0^L exp(-i tau_j r) T_{a,x}(r) dE_x^leftarrow(r).      (BTF)
```

This identity is exact: it is only the definition of the packet coordinates of `Y_x(tau_j)`.

## 1. Exact finite equivalence

The remaining upper-bound half of the S-FIN scale split is:

```text
||P_{B,x}Y_x(tau_j)|| <= C sqrt(x).                          (BUP)
```

Since `Q_x` is orthonormal,

```text
||P_{B,x}Y_x(tau_j)||^2
 = sum_{a=1}^b |<Y_x(tau_j),e_a>|^2
 = sum_{a=1}^b
   |int_0^L exp(-i tau_j r)T_{a,x}(r)dE_x^leftarrow(r)|^2.
```

Therefore `(BUP)` is equivalent to the finite bad-test square bound:

```text
sum_{a=1}^b
 |int_0^L exp(-i tau_j r)T_{a,x}(r)dE_x^leftarrow(r)|^2
 <= C^2 x                                                     (BTB)
```

uniformly for finite root-height windows.

This is stronger and cleaner than a componentwise formulation, because the left side is invariant
under unitary changes of the bad basis.

## 2. Why this is the right target

A bound of the form:

```text
int f(r)dE_x^leftarrow(r)=O(sqrt(x))
```

for all smooth tests `f` would be too strong and would re-enter the usual explicit-formula wall.
The point of `(BTB)` is that only finitely many tests are present, and they are not arbitrary.  They
are generated by the shorted Loewner-Feshbach map:

```text
mass row + first q right singular vectors of R_x.
```

Thus the next theorem should not be a general discrepancy theorem.  It should be a shorting
orthogonality theorem:

```text
shorted bad tests at finite CCM roots see only square-root endpoint energy.
```

In the notation of E72.124, the intended final scale separation is:

```text
full packet frame:  ||Y_x(tau_j)|| >= c sqrt(x)L,
bad tests:          sum_a |<Y_x(tau_j),e_a>|^2 <= Cx.
```

Together these imply:

```text
||P_BY||/||Y||=O(1/L),
```

hence Gram saturation and scalar WRL annihilation.
