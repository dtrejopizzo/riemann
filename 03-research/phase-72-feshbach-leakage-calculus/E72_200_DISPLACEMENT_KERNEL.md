# E72.200 - Displacement kernel diagnostic

## Purpose

E72.199 showed that the high-block cubic sign is entirely non-resonant. This note inspects the
largest non-resonant displacement classes.

The target object is the class sum

```text
C_r(R) =
sum_{prod n_i^eps_i = R}
Tr(A_{n_1}^{eps_1}...A_{n_r}^{eps_r}).
```

For the high block and `r=3`, the resonant class is absent, so the sign is controlled by these
`C_3(R)`.

## Reading

Two structural features appear.

First, the dominant classes occur in reciprocal pairs:

```text
C_r(R)=C_r(1/R)
```

within numerical precision. This is the expected adjoint symmetry of signed Green-words.

Second, many dominant classes are effectively single-displacement classes:

```text
R=p or R=1/p.
```

That means the non-resonant contribution is not featureless. It is governed by an even displacement
kernel in `log R`, sampled on the multiplicative lattice generated by prime powers.

## New theorem shape

The next proof should not estimate individual words. It should seek a kernel representation

```text
C_r(R) = W_r(log R) * M_r(R),
```

where:

```text
W_r(log R)  is an even Green-translation kernel from the model complement,
M_r(R)      is a finite multiplicative correlation of Lambda(n)n^{-1/2}.
```

For the high-block cubic sign, the desired statement is:

```text
sum_{R != 1} C_3(R) <= 0.
```

This is the first genuinely sharp non-resonant target: it keeps the Green phase and the exact
multiplicative grouping.

## Status

Still open, but the target is now a displacement-kernel inequality rather than an unstructured
prime-word inequality.
