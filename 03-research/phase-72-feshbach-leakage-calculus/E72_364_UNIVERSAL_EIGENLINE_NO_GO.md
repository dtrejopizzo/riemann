# E72.364 - Universal eigenline no-go

## 1. Purpose

E72.363 rewrites the simple-window `CFIR-DIV` condition as an eigenline statement:

```text
P_k^perp(KWin k-tail)=0.
```

It is tempting to try to prove this by showing that the kernel matrix `KWin` preserves many lines.
This note proves that such a universal route is impossible.

## 2. Universal eigenline hypothesis

Suppose that, for every finite shifted window `A={a_1,...,a_N}`, the matrix

```text
KWin_A=(K_L(a_i,a_j))
```

preserved the line generated by the relevant vector `k` for structural reasons independent of `Xi`.
In the strongest node-blind form, this would imply that generic shifted controls pass the eigenline
test.

E72.363 shows they do not:

```text
shift relPerp ~= 0.13 to 0.43.
```

Thus the node-blind eigenline property is numerically false.

## 3. Structural obstruction from the Cauchy block

There is also a purely analytic obstruction.  On a maximal positive-real-part off-line cluster,
E72.324--E72.325 prove that the leading block of the interpolation matrix is

```text
-diag(e^(a_iL)) C_A,
        C_A(i,j)=1/(a_i+a_j),
```

or its confluent Cauchy version.  The Cauchy matrix `C_A` is invertible for distinct representatives
with `Re(a_i)>0`.

An invertible Cauchy matrix does not preserve all lines.  Therefore no universal rank-one or
line-preserving identity can hold for `KWin_A` on arbitrary windows of size `N>=2`.

## 4. Consequence

The eigenline relation required by `CFIR-DIV` cannot be:

```text
KWin alone has a universal invariant line.
```

It must be:

```text
the actual finite Feshbach residue vector k=J_TG_x
satisfies a special Xi-divisor equation
whose leading Cauchy block is invertible.
```

This returns the problem to the correct equation:

```text
H_T(L)J_TG_x + J_TTail_T^M = O_T(L^B).
```

The eigenline formulation is still useful, but only as a certificate for the exact solution vector,
not as a standalone kernel theorem.

## 5. Updated target

The proof must now show:

```text
Xi-divisor eigenline:
for the actual residue vector k=J_TG_x on an Xi-zero window,
P_k^perp(KWin k-tail)=O_T(L^B).
```

This has to be derived from the coupled Feshbach/explicit-formula identity, not from a universal
property of `K_L`.

## 6. Status

```text
proved: universal KWin eigenline preservation is false;
proved: leading Cauchy invertibility blocks a rank-one kernel explanation;
open: prove the Xi-divisor eigenline for the actual Feshbach residue vector.
```
