# E71.4 -- Riesz-window stability results

**Date:** 2026-07-07.
**Script:** `E71_4_riesz_window_stability.py`.

## Wide-window result

The first attempt used Voronoi windows between consecutive target zeros. This was too coarse.

For true zeta at `lambda=5`, `N=40`, the first five windows had ranks

```text
3, 3, 2, 1, 2
```

even though each target zero had a nearby CCM root at about `1e-7` error.

Reading: finite CCM spectra contain extra roots inside the same large zero gaps. Therefore the right
Riesz diagnostic is not "one root in the whole gap between neighboring zeta zeros".

## Corrected diagnostic

Use small local contours around each target zero:

```text
|t - gamma_j| < r.
```

For diagnostics we may center these contours at the known zeros. In a final proof the contours must be
generated by a zero-independent analytic limiting argument; this script is only testing the mechanism.

The expected pattern is:

```text
true zeta:          local rank one near each gamma_j;
planted off-line:   local ranks fail or retain displaced floors.
```

With radius `r=1e-3`, the observed local statuses were:

```text
zeta true:
N=14: LLL--
N=18: LLLLL
N=20: LLLL-
N=24: LLL-L
N=30: L--LL
N=36: -LLLL
N=40: LLLLL

planted off-line:
N=14: -----
N=18: -----
N=20: -----
N=24: -----
N=30: -----
N=36: -----
N=40: -----
```

Here `L` means exactly one root in the micro-contour. The planted perturbation has roots displaced by
roughly `1e-2` or more, so it misses every `1e-3` local contour.

## Interpretation

E71.4 corrects E71.3's operator language:

```text
nearest-root matching: too noisy;
wide Riesz windows:    too coarse;
micro Riesz contours:  right finite diagnostic.
```

The theorem needed for closure is therefore local Riesz convergence, not global gap occupancy:

```text
for each compact height window and each limiting zero contour,
the CCM Riesz projection converges with zero-independent error bounds.
```

This is still not a proof of RH, but it is the correct form of the CAND-1 convergence theorem.
