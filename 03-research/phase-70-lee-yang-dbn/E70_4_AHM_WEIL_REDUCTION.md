# E70.4 — AHM reduces to Weil positivity on Cauchy test squares

**Date:** 2026-07-07.
**Role:** push the Arithmetic Herglotz Majorization route as far as it can go without fabricating a
proof of RH.

## Verdict

The master inequality

```text
AHM:  K_P >= -K_A
```

is exactly Weil positivity, restricted to the Cauchy/Pick test cone generated by upper-half-plane
resolvent vectors. This is not a failure: it is the cleanest localization of the remaining problem.
But it means AHM is not currently a smaller theorem than `Omega_7`; it is the gauge-free Weil-hard core.

The gain is that the hard statement now has a very explicit form:

```text
Euler-prime quadratic form >= archimedean drag
for every Cauchy square |sum_j c_j/(u-z_j)|^2.
```

This is the exact arithmetic inequality that would close `Omega_7`.

## 1. Variables and sign

Let

```text
F(z) = xi(1/2 + i z),
N(z) = -F'(z)/F(z).
```

If all zeros `gamma` of `F` are real, then

```text
N(z) = - sum_gamma 1/(z-gamma) + harmless affine/regular terms,
```

and each real zero contributes the positive Pick atom

```text
K_gamma(z,w)
 = (-1/(z-gamma) + 1/(conj(w)-gamma)) / (z-conj(w))
 = 1/((z-gamma)(conj(w)-gamma)).
```

Thus `N` is Nevanlinna exactly when the zero measure is supported on the real axis.

## 2. Cauchy test vectors

For points `z_1,...,z_m` in the upper half-plane and coefficients `c_j`, define

```text
phi(u) = sum_j c_j/(u-z_j).
```

The Pick quadratic form is

```text
Q_N(c;z_j)
 = sum_{i,j} c_i conj(c_j)
   (N(z_i)-conj(N(z_j))) / (z_i-conj(z_j)).
```

If RH holds, the zero-side representation is

```text
Q_N(c;z_j) = sum_gamma |phi(gamma)|^2 >= 0.
```

Without RH, the same expression is the Krein-Langer signed index form; off-line zeros contribute
negative squares. This is the gauge-free version of the `A_N-P_lambda` detector.

## 3. Explicit-form split

The logarithmic derivative splits as

```text
N = N_A + N_P,
```

where `N_A` is the polar/Gamma term and `N_P` is the analytically continued prime/Euler term.

Correspondingly,

```text
Q_N = Q_A + Q_P.
```

AHM says

```text
Q_P(phi) >= -Q_A(phi)
```

for every Cauchy test `phi`.

This is exactly the explicit formula read as a positivity statement:

```text
prime side + archimedean side = zero side >= 0.
```

The novelty is not the equality; the equality is the explicit formula. The required new mathematics is
the positivity of the right-hand zero-side without knowing that the zeros are real.

## 4. Why there is no termwise Euler proof

In the upper `z` half-plane,

```text
s = 1/2 + i z
```

has

```text
Re(s) = 1/2 - Im(z) < 1/2.
```

The Euler series

```text
-zeta'/zeta(s) = sum_n Lambda(n)n^{-s}
```

does not converge there. Therefore `N_P` in AHM is not a literal positive prime sum in the domain where
Nevanlinna positivity must be proved. It is an analytic continuation of the Euler object across the
critical strip.

So any proof of AHM must cross exactly the classical wall:

```text
Euler product in Re(s)>1
  -> analytic continuation / explicit formula
  -> positivity in the opposite half-plane.
```

That crossing is Weil positivity.

## 5. Cauchy-square Weil criterion

Define the Weil functional `W` on admissible test functions by the explicit formula:

```text
W(g) = zero-side(g)
     = archimedean(g) + prime(g).
```

For the special test family

```text
g(u) = |phi(u)|^2,
phi(u) = sum_j c_j/(u-z_j),  Im z_j>0,
```

one has

```text
W(|phi|^2) = Q_N(c;z_j).
```

Therefore:

```text
AHM
<=> W(|phi|^2) >= 0 for every Cauchy rational phi.
```

Since Cauchy rational functions are dense in the Hardy/Nevanlinna test space, this is the same
positivity cone as Weil's criterion, expressed in the terminal Pick coordinates.

## 6. Exact closure lemma

The closure of `Omega_7` is equivalent to the following lemma.

### Cauchy-Weil Positivity Lemma

For every finite upper-half-plane Cauchy rational function

```text
phi(u)=sum_j c_j/(u-z_j),
```

the explicit-form functional satisfies

```text
W(|phi|^2) >= 0.
```

Equivalently,

```text
Q_P(phi) >= -Q_A(phi).
```

If this lemma is proved from Euler structure, then:

```text
Cauchy-Weil positivity
  => K_Nev >= 0
  => Im(-F'/F)>=0
  => F in Laguerre-Polya
  => Lambda <=0
  => Omega_7/RH.
```
