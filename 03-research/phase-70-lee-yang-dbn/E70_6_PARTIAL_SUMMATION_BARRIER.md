# E70.6 — Partial-summation attack on the Hardy-Euler inequality

**Date:** 2026-07-07.
**Role:** attack the autocorrelation inequality directly and isolate the exact arithmetic cancellation
required.

## Starting inequality

The Hardy-Euler form of `Omega_7` is:

```text
2 Re sum_{n>=2} Lambda(n)n^{-1/2} C_f(log n) <= A[f]
```

for every finite exponential sum

```text
f(t)=sum_j c_j e^{i z_j t},   Im z_j>0,
```

where

```text
C_f(a)=int_0^infty f(t+a)conj(f(t))dt.
```

Equivalently, in the additive shift model of P52:

```text
<T_Lambda f,f> <= A[f],
T_Lambda = sum_{n>=2} Lambda(n)n^{-1/2}(S_log n+S_log n^*).
```

This is exactly the terminal spectral form of `Omega_7`.

## Average-prime cancellation

Write the prime measure in additive coordinates:

```text
dPsi(t) = sum_{n>=2} Lambda(n) delta(t-log n).
```

The weight `n^{-1/2}` becomes `e^{-t/2}`. The prime form is

```text
P[f] = 2 Re int_0^infty e^{-t/2} C_f(t) dPsi(t).
```

Split

```text
dPsi(t) = e^t dt + dE(t),
E(t)=psi(e^t)-e^t.
```

Then

```text
P[f] = P_avg[f] + P_err[f],
```

where

```text
P_avg[f] = 2 Re int_0^infty e^{t/2} C_f(t) dt,
P_err[f] = 2 Re int_0^infty e^{-t/2} C_f(t) dE(t).
```

The pole/archimedean main term cancels `P_avg[f]`. This is the same cancellation as the pole-minus-prime
renormalization in the Laguerre/Li form of P52.

So the terminal inequality is equivalent to bounding the error term:

```text
P_err[f] <= A_rem[f],
```

where `A_rem[f]` is the finite Gamma/polar remainder after the average-prime cancellation.

## Partial summation

For smooth enough `C_f`, integrate by parts:

```text
P_err[f]
 = - 2 Re int_0^infty E(t) d/dt(e^{-t/2} C_f(t)) dt
```

up to endpoint terms, which vanish for the exponentially decaying Hardy tests.

In multiplicative variables `x=e^t`, this is

```text
P_err[f]
 = - 2 Re int_1^infty (psi(x)-x)
       d/dx( x^{-1/2} C_f(log x) ) dx.
```

This is the exact arithmetic face of the problem.

## Why standard estimates fail

An absolute bound gives

```text
|P_err[f]|
 <= int_1^infty |psi(x)-x|
      |d/dx(x^{-1/2}C_f(log x))| dx.
```

To dominate `A_rem[f]` uniformly for all Hardy autocorrelations, this requires square-root-scale
cancellation of the prime error against every oscillatory kernel generated by `C_f`.

This is not available from the prime number theorem. It is RH-strength:

```text
psi(x)-x = O(x^{1/2+epsilon})
```

is already essentially equivalent to zeros on the critical line. The Laguerre version in P52 says the
same thing differently: the kernel reaches `x ~ exp(4N)`, so the required cancellation is over
exponentially long ranges.

## The exact missing estimate

The proof of `Omega_7` by this route is equivalent to the following uniform oscillatory estimate.

### Hardy-prime cancellation estimate

For every finite Hardy exponential sum `f`,

```text
- 2 Re int_1^infty (psi(x)-x)
       d/dx( x^{-1/2} C_f(log x) ) dx
 <= A_rem[f].
```

This estimate is the fully expanded form of AHM.

## Why this is not a loss

The estimate has the right falsifiers:

- If an off-line zero exists, then `psi(x)-x` has a term of size `x^beta/beta`, `beta>1/2`, and one can
  choose a Hardy autocorrelation resonant with that term; the inequality fails.
- If all zeros are on the line, the error terms are only critical-line oscillations and the Weil form is
  positive.

Thus the estimate is exactly RH in prime-error language.

## Failed proof attempt: termwise domination

One might hope to show

```text
|P_err[f]| <= A_rem[f]
```

from decay of `C_f` alone. This fails for the same reason as E67.4:

```text
decay/absolute value -> incoherent ceiling S_abs,
true operator        -> marginal by phase cancellation.
```

The proof must use signed oscillation of `psi(x)-x`, not just size.

## Current status

The autocorrelation inequality is now reduced to a single prime-error cancellation estimate. This is a
clean work surface, but it is still RH-strength:

```text
Omega_7
<=> Hardy-Euler autocorrelation inequality
<=> Hardy-prime cancellation estimate
<=> critical-line control of psi(x)-x.
```

I do not have a non-circular proof of this estimate. Any proof here would be a proof of RH.
