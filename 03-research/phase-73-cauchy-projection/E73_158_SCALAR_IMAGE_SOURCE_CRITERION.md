# E73.158 - Scalar image source criterion

Date: 2026-07-12.

## 1. Purpose

E73.157 identifies the final finite source as

```text
SRC_b=<xi_L,S_b>.
```

The old image route is dangerous if stated as norm membership:

```text
S_b in Image(E_L)
```

because exact image membership is equivalent to the desired eigenline
orthogonality.  E73.020 already corrected this: the only scalar-relevant
quantity is the `xi_L` component of the source after removing a structured
image.

This note states the usable criterion.

## 2. Structured image

Let

```text
E_L=H_L-mu_L I.
```

Let `P_L` be the structured rational/divided-difference primitive space
generated by the kernels

```text
DD_L(-gamma,d)/(d^2+beta^2)^r,
DD_L(-gamma,d),
d^2 DD_L(-gamma,d),
```

for the critical nodes `gamma` and off-line projection nodes `beta` in the
finite Cauchy block.

Define the structured image

```text
I_L:=Image(E_L P_L).
```

For each source `S_b`, write the orthogonal decomposition

```text
S_b = P_{I_L}S_b + R_b.
```

Since `E_L xi_L=0`, every vector in `I_L` is orthogonal to `xi_L`, hence

```text
<xi_L,S_b>=<xi_L,R_b>.                                (1)
```

Therefore `Strong-CC-PROJ` follows from the scalar residual estimate

```text
e^(Re b L)|<xi_L,R_b>| <= L^B.                        (2)
```

This is weaker than norm-smallness of `R_b` and is the only statement needed.

## 3. Non-tautology condition

The projection space `I_L` must be fixed structurally before seeing `S_b`:

```text
I_L=Image(E_L P_L)
```

with `P_L` generated by the explicit rational/divided-difference family, not
by the source itself and not by a full pseudoinverse of `E_L`.

Otherwise (1) is only a restatement of `SRC_b`.

## 4. Proof target

The analytic theorem to prove is:

```text
SCALAR-IMAGE-SRC:
For the natural-height Cauchy projection window and structured primitive
space P_L,

max_b e^(Re b L)|<xi_L,(I-P_{I_L})S_b>| <= L^B.
```

Together with E73.157:

```text
SCALAR-IMAGE-SRC
=> Strong-CC-PROJ / NAT-SRC
=> NAT-PROJ
=> scalar WRL
=> Omega_7.
```

## 5. What remains to prove

The numerical image rank is small, but a proof needs a symbolic description
of the low-rank image:

```text
E_L P_L = span of a finite endpoint-critical image family
```

plus an explicit estimate of the scalar residual against `xi_L`.

This is not rowspace or pseudoinverse closure.  It is a finite structured
coborder theorem in the exact source coordinates.
