# E73.141 - Local Divisor Row Theorem

Date: 2026-07-12.

## 0. Audit position

This note deliberately does not use the old determinant route.

Previous phases already showed:

```text
global determinant identification -> CCM convergence -> RH frontier,
global strong resolvent convergence -> Phase 65 wall,
adjugate/cofactor compatibility -> finite certificate but not a forcer.
```

The surviving Phase 73 mechanism is local: a Cauchy row at a critical height
is controlled by a finite divisor identity and a mesh-tail estimate.

## 1. Notation

Let

```text
d_n = 2 pi n/L,
H_0(u)=sum_n xi_n/(u+i d_n).
```

For a critical row `gamma`, the real Cauchy value used in `CSV_17` is the
same object up to the harmless sign convention:

```text
C_x(-gamma)=sum_n xi_n/(-gamma-d_n)
           = -i H_0(-i gamma).
```

Thus

```text
CSV_17 <=> |H_0(-i gamma)| <= C L^-17
```

for every natural-height critical row.

The normalized row version is

```text
|<xi,e_gamma>| <= C L^{-17-p_gamma},
||k_gamma|| <= C L^{p_gamma}.
```

## 2. Exact complete-mesh divisor theorem

E73.075 proves, for the complete-mesh packet,

```text
Pair_z^infty(w)=A_L(z,w)H_0(w)+B_L(z,w)H_0(-w).       (1)
```

This is an exact algebraic identity.  It comes only from partial fractions
and the mesh identity `exp(i d_n L)=1`.

Consequently, at a symmetric finite Cauchy root pair,

```text
H_0(w)=H_0(-w)=0,
```

the paired packet vanishes identically:

```text
Pair_z^infty(w)=0.
```

This is the first non-tautological divisor mechanism in Phase 73: it does
not assume global convergence to `Xi`, and it does not use the zeros as
spectral input.  It is a finite identity for the current generated by
`(d_n,xi_n)`.

## 3. Local row reduction

Fix a critical height `gamma` and set

```text
w_gamma=-i gamma.
```

Assume the following local divisor estimate:

```text
LDIV_17:
|H_0(w_gamma)|+|H_0(-w_gamma)| <= C L^-17
```

for all natural-height critical rows in the standard boxes.

Then `CSV_17` follows immediately from §1:

```text
|C_x(-gamma)| = |H_0(w_gamma)| <= C L^-17.
```

Together with E73.123 and E73.120, this gives Uniform GATE-73.

The point is that `LDIV_17` is not a global CCM convergence statement.  It
is only a finite list of local divisor values in the Phase 73 boxes.

## 4. How to prove LDIV without circularity

The packet equation from E73.070 has the form

```text
Gap_x G_x(z)=ZPacket_x(z)-TailPacket_x(z).
```

After pairing `w` and `-w`, the complete-mesh part factors by (1).  Therefore
the only non-complete terms are:

```text
finite mesh tail,
endpoint correction,
standard-box FAR3 coefficients.
```

The non-circular proof target is:

```text
LOCAL-DIVISOR-FORCING.
For w=w_gamma in the critical row window,

|H_0(w)|+|H_0(-w)|
 <= C (
      |Pair_z^M(w)-Pair_z^infty(w)|
      + |endpoint_z(w)|
      + |standard-box residual_z(w)|
    )
```

with the right side `O(L^-17)` after the E73.120 finite manifest and the
E73.123 elementary envelopes.

This is the analytic theorem that remains.  It is local, finite, and
rowwise.  It does not ask for:

```text
Phi_N -> Xi globally,
strong resolvent convergence,
positivity of a Weil form,
or determinant identification.
```

## 5. Why this is stronger than CCC

The critical cell cancellation

```text
<e,W02 xi>-<e,Prime xi>-<e,WR xi>
```

is already small at numerical precision because `H xi=mu xi`.

That verifies consistency but does not force

```text
<e,xi>
```

to be small.

`LDIV_17` targets `H_0(w)` directly.  It is exactly the missing evaluation
smallness, not the eigen-equation multiplied by a tiny eigenvalue.
