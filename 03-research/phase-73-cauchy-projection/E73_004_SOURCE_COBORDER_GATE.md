# E73.004 - Source coborder gate

## 1. Purpose

E73.003 reduces the remaining projection theorem to

```text
SRC_b=<xi,S_b>.
```

Since `xi` is the pole-even eigenvector of the finite CCM/Feshbach operator, the natural way to prove
`SRC_b` small is to show that `S_b` is a coborder for the adjoint eigen-equation.

This note states the exact finite gate.

## 2. Eigenline orthogonality

Let the pole-even finite operator be

```text
C_E xi = mu xi,
||xi||=1.
```

For any vector `Y`,

```text
<xi,(C_E-mu I)Y>=0.                                  (1)
```

Therefore, if the source admits a decomposition

```text
S_b=(C_E-mu I)Y_b + R_b,                              (2)
```

then

```text
<xi,S_b>=<xi,R_b>.                                   (3)
```

Thus `NAT-SRC` follows if

```text
e^(Re b L)|<xi,R_b>| <= L^B.                         (4)
```

## 3. Exact finite formulation

Let `P_xi` denote the orthogonal projection onto the pole-even eigenline:

```text
P_xi S=<xi,S>xi.
```

Equation (2) is solvable with small `R_b` if and only if the eigenline component of `S_b` is small:

```text
||P_xi S_b||=|<xi,S_b>|.
```

So this is not yet a proof.  The useful finite criterion is a **structured** coborder:

```text
S_b=(C_E-mu I)Y_b + R_b
```

where `Y_b` is built from the same rational/divided-difference class as `S_b`, and `R_b` is an endpoint
term whose size can be bounded by the natural-height estimates of Phase 72.

## 4. Coborder target

Define the class `RAT-DD_L(O,K)` as the finite span of kernels

```text
d -> DD_L(-gamma,d) * rational(d;O,K),
```

where `gamma` runs through critical heights in `K_L` and the rational factor is generated by the
Cauchy interpolation weights `Pi(b,k)`.

The proof-facing target is:

```text
SRC-COB:
For every off-line node b in O_L, there exists Y_b in RAT-DD_L(O,K) such that

   S_b=(C_E-mu I)Y_b + R_b

and

   e^(Re b L)|<xi,R_b>| <= L^B.
```

Then:

```text
SRC-COB => NAT-SRC => NAT-PROJ.
```

## 5. Why this is the right next object

Generic approximation of `S_b` is useless: the component along `xi` is exactly the quantity to prove.
The only non-circular route is to use the explicit operator identity

```text
C_E = W02 - WR - Prime - e_pole I
```

and show that applying it to a rational divided-difference primitive reproduces `S_b` up to the
already closed endpoint/high-frequency remainders.

This is the finite analogue of a cohomological coboundary: the source is invisible to the eigenline
because it is an operator boundary.
