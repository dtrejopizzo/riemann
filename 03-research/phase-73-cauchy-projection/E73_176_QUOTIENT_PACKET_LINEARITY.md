# E73.176 - Quotient packet linearity

Date: 2026-07-14.

## 1. Purpose

E73.175 reduces `QG` to a boundary estimate for three scalar packets:

```text
B_{r,L}(y)=a_r(L)Q_L(y)xi_L.
```

This note identifies these packets with linear combinations of the transformed
Phase 72 packets.

## 2. Transformed rows

For a shifted parameter `z`, Phase 72 uses the transformed Cauchy row:

```text
R_z(d_m) = (1-e^(zL))/(iz-d_m).
```

At a critical node

```text
z_j=i gamma_j,
```

this becomes:

```text
R_{z_j}(d_m)
= (1-e^(i gamma_j L))/(-gamma_j-d_m),
```

which is exactly the stable Cauchy row used in E73.171--E73.175.

## 3. Quotient rows

Let

```text
q_1(L),q_2(L),q_3(L)
```

be an orthonormal basis of the quotient `I_L^perp` in the five-dimensional
critical `cauchy0` space.  Then:

```text
a_r(L) = sum_{j=1}^5 conjugate(q_r(L)_j) R_{i gamma_j}.
```

Therefore:

```text
B_{r,L}(y)
= sum_{j=1}^5 conjugate(q_r(L)_j) B_{i gamma_j,L}(y),
```

where `B_{z,L}` is the transformed packet from Phase 72.

## 4. Consequence

All algebraic packet identities from Phase 72 may be applied linearly to the
three quotient packets.  In particular, the endpoint-sensitive Abel identity,
the packet Mellin transform, and the symmetric zero-pair cancellation have
quotient versions obtained by summing the five critical-node equations with
coefficients `conjugate(q_r(L)_j)`.

## 5. What is genuinely new

Phase 72 aimed to prove polynomial control of:

```text
G_x(z).
```

That is not enough here.  E73 needs the stronger quotient cancellation:

```text
sum_{j=1}^5 conjugate(q_r(L)_j) G_x(i gamma_j)
= O(L^B e^(-alpha L)).
```

Equivalently:

```text
B_{r,L}(0)=O(L^B e^(-alpha L)).
```

Thus E73 does not repeat Phase 72.  It takes the Phase 72 transformed packet
calculus and applies it after quotienting out the two directions generated by
the finite rational local action.

## 6. Proof-facing endpoint

The next theorem should be the quotient version of the transformed packet
equation:

```text
QTPW:
For r=1,2,3, the linear combination of the five critical-node TPW equations
with coefficients conjugate(q_r(L)_j) forces

|B_{r,L}(0)| <= L^B e^(-alpha L).
```

In Phase 72 language, this is not merely `PW-Cauchy`; it is a critical-node
quotient annihilation theorem.
