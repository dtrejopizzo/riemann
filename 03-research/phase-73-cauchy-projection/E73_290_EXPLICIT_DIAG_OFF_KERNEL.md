# E73.290 - Explicit diagonal/off-diagonal kernel

Date: 2026-07-14.

## 1. Purpose

E73.289 left the endpoint as the dual identity

```text
<c^off_{a,w},W'>+<c^diag_{a,w},U'>=O_M(L^(-M)),
U'=W'-V'/L.
```

This document removes one layer of coefficient bookkeeping.  The whole
diagonal/off-diagonal pairing is a single finite mesh pairing

```text
sum_b eta_b K_b,
eta=R_w xi_L.
```

The point is to locate the exact residual vector `K` that must be killed by
the Gamma-prime eigenline equation.

## 2. Definitions

Let the CCM mesh be indexed by `j`, with integer labels `n_j` and frequencies

```text
d_j=2*pi*n_j/L.
```

For a Cauchy row `r_j=q_a(d_j)` define the even and odd closed-weight rows

```text
W^odd_j = W'_{d_j}-W'_{-d_j},
U^even_j = U'_{d_j}+U'_{-d_j}.
```

The explicit kernel is

```text
K_b
= r_b U^even_b
  + sum_{j != b} r_j (W^odd_j-W^odd_b)/(2 i pi (n_b-n_j)).
```

Call this formula `K-DIAGOFF`.

## 3. Algebraic proof

The exact CCM cell expansion has two sectors.

Diagonal cells contribute

```text
c^diag_{d_b}  += r_b eta_b,
c^diag_{-d_b} += r_b eta_b.
```

Therefore

```text
<c^diag,U'> = sum_b eta_b r_b (U'_{d_b}+U'_{-d_b})
             = sum_b eta_b r_b U^even_b.
```

For an off-diagonal cell `(j,b)`, `j != b`, E73.288 gives

```text
c^off_{d_j}  += r_j eta_b /(2 i pi (n_b-n_j)),
c^off_{-d_j} +=-r_j eta_b /(2 i pi (n_b-n_j)),
c^off_{d_b}  +=-r_j eta_b /(2 i pi (n_b-n_j)),
c^off_{-d_b} += r_j eta_b /(2 i pi (n_b-n_j)).
```

Pairing with `W'` yields

```text
r_j eta_b /(2 i pi (n_b-n_j))
[(W'_{d_j}-W'_{-d_j})-(W'_{d_b}-W'_{-d_b})]

= eta_b r_j (W^odd_j-W^odd_b)/(2 i pi (n_b-n_j)).
```

Summing over `j != b` and adding the diagonal sector proves

```text
<c^off,W'>+<c^diag,U'> = sum_b eta_b K_b.          (K-PAIR)
```

This is an exact finite identity.  It uses no least-squares source
coordinates and no rational basis saturation.

## 5. Consequence

The sharp endpoint is now:

```text
KERNEL-FREQ-DIV:
sum_b (R_w xi_L)_b K_{a,w,L}(b)=O_M(L^(-M)),
```

with `K` given by `K-DIAGOFF`.

This is strictly more explicit than `PAIR-DUAL-FREQ-DIV`.  It separates the
problem into:

```text
1. closed weights W^odd and U^even;
2. the finite Hilbert-transform-like mesh operator
   (f_j-f_b)/(n_b-n_j);
3. the projected eigenvector eta=R_w xi_L.
```

The next theorem cannot be a Cauchy-plane annihilation theorem.  It must show
that this residual kernel is generated by the Gamma-prime eigenline relation,
or by an equivalent finite commutator identity for the mesh Hilbert transform.
