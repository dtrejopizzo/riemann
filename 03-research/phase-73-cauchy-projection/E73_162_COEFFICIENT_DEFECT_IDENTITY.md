# E73.162 - Coefficient defect identity

Date: 2026-07-12.

## 1. Why this note is needed

E73.161 identifies the current endpoint as:

```text
S_A = (H_L-mu_L I)Y_A + R_A,
e^(alpha L)|<xi_L,R_A>| <= L^B.
```

But E73.004, E73.020, and E73.125--E73.126 already warn that this statement is
tautological if `Y_A` is chosen by a free projection onto
`Image(H_L-mu_L I)`.  The scalar

```text
<xi_L,S_A-(H_L-mu_L I)Y>
```

is independent of `Y`.  Therefore no proof can come from ambient least
squares, pseudoinverses, or generic rowspace distance.

The only non-circular version is coefficient-level:

```text
choose Y_A from a fixed rational module,
compute (H_L-mu_L I)Y_A by the CCM cell formula,
and show that its principal coefficients equal the source coefficients.
```

The residual must then be an explicitly named endpoint residual, not a
projection residual.

## 2. Rational modules

On the pole mesh

```text
d_n=2 pi n/L,
```

E73.018 gives

```text
DD_L(-gamma,d_n)
= (exp(i gamma L)-1)/(gamma+d_n).
```

Thus all source vectors are finite Cauchy-rational functions of `d`.

For a natural-height off-line Hermite cluster `A` and critical window `K_L`,
define the primitive module

```text
P_{A,K}
= span{
    1/((d+gamma)(d^2+beta^2)^r) : gamma in K_L, beta in B(A), r=0,1,2
  }
  + span{ d^2/(d+gamma) : gamma in K_L }.
```

Here `B(A)` is the finite set of beta parameters generated by the Hermite
cluster.  For a simple cluster centered at `a=alpha+i tau`, one may take
`beta` from the reflected denominator `|a|` and its fixed confluent
companions; for a multiple cluster `B(A)` is the finite Hermite list fixed
by E73.026--E73.030.

Define the source module

```text
S_{A,K}
= span{
    1/((d+gamma)(d^2+beta^2)^r) : gamma in K_L, beta in B(A), r=0,1,2,3
  }
  + span{ d^2/(d+gamma), d^4/(d+gamma) : gamma in K_L }.
```

The source vector from NAT-SRC is an element of `S_{A,K}`:

```text
S_A(d)
= sum_{k=i gamma in K_L} Pi_conf(A,k) DD_L(-gamma,d).
```

Equivalently, in source coordinates,

```text
s_A = coefficient vector of S_A in S_{A,K}.
```

## 3. The explicit coefficient action

E73.017 decomposes the coupled finite CCM cell as

```text
H_L-mu_L I = A_0(D) + L_A(D,J),
```

where `A_0(D)` is diagonal on the pole mesh and `L_A(D,J)` is one finite
Loewner displacement:

```text
L_A(D,J)_{mn} = (F(d_m)-F(d_n))/(d_m-d_n)
```

up to the diagonal removable convention and the already isolated endpoint
cell corrections.

For each primitive basis vector `p_j(d)` in `P_{A,K}`, expand

```text
(H_L-mu_L I)p_j
= sum_i B_L(A,K)_{ij} s_i(d) + r_j(d),
```

where `s_i(d)` runs over the fixed basis of `S_{A,K}` and `r_j` belongs to
the Phase 72 endpoint residual class.

This defines a finite coefficient matrix:

```text
B_L(A,K): P_{A,K}^{coeff} -> S_{A,K}^{coeff}.
```

The entries of `B_L(A,K)` are obtained by partial fractions from the two
elementary identities:

```text
A_0(d) / ((d+gamma)(d^2+beta^2)^r)
```

and

```text
(F(d)-F(e))/(d-e) * 1/((e+gamma)(e^2+beta^2)^r).
```

No eigenvector, zero of `Xi`, or final scalar pairing is used to compute
`B_L`.

## 4. Coefficient defect

Define the coefficient defect of the source by

```text
Def_L(A)
= dist_coeff(s_A, Image B_L(A,K)),
```

but this distance is not an ambient Hilbert projection.  It means:

```text
there exists a coefficient vector y_A such that

s_A - B_L(A,K)y_A = eps_A,
```

where `eps_A` is supported only on the explicitly allowed endpoint residual
slots.

Equivalently,

```text
S_A = (H_L-mu_L I)Y_A + R_A,
```

with

```text
Y_A = sum_j y_A(j)p_j,
R_A = endpoint residual represented by eps_A and by the r_j terms.
```

This is the first non-tautological form of the defect identity because:

```text
1. `Y_A` is fixed by coefficient equations in a prescribed rational module;
2. the residual class is fixed before pairing with xi_L;
3. no free projection onto xi_L^perp is allowed.
```

## 5. The theorem that would close the Phase 73 gate

The load-bearing theorem is:

```text
COEFF-DEF:
There exist B and L0 such that for every L>=L0 and every natural-height
off-line Hermite cluster A with real part alpha>0, there is a coefficient
vector y_A in P_{A,K}^{coeff} satisfying

s_A - B_L(A,K)y_A = eps_A
```

with endpoint residual `R_A` obeying

```text
e^(alpha L)|<xi_L,R_A>| <= L^B.
```

Then:

```text
COEFF-DEF
=> DATA-HERM / NAT-SRC
=> NAT-PROJ
=> scalar WRL
=> Omega7.
```

## 6. Proof of the implication

Assume `COEFF-DEF`.  Let

```text
Y_A = sum_j y_A(j)p_j.
```

By definition of `B_L(A,K)`,

```text
(H_L-mu_L I)Y_A
= sum_i (B_L y_A)_i s_i + R_A^{cell},
```

where `R_A^{cell}` is in the Phase 72 endpoint residual class.

Since

```text
s_A - B_Ly_A = eps_A,
```

the corresponding mesh vectors satisfy

```text
S_A = (H_L-mu_L I)Y_A + R_A,
```

with `R_A` the sum of the endpoint vector represented by `eps_A` and
`R_A^{cell}`.

Because

```text
(H_L-mu_L I)xi_L=0
```

and the operator is Hermitian,

```text
<xi_L,(H_L-mu_L I)Y_A>=0.
```

Therefore

```text
<xi_L,S_A>=<xi_L,R_A>.
```

The residual bound in `COEFF-DEF` gives

```text
e^(alpha L)|<xi_L,S_A>| <= L^B,
```

which is `NAT-SRC` in Hermite source coordinates.  E73.031--E73.033 identify
this with `DATA-HERM`, and E73.001--E73.003 identify it with `NAT-PROJ`.
The Phase 72 chain then gives scalar WRL and Omega7.

## 7. What remains genuinely open

The open work is now explicit:

```text
1. construct B_L(A,K) symbolically from the coupled CCM cell;
2. prove the source coefficient vector s_A lies in Image B_L modulo endpoint
   slots;
3. prove the endpoint slots pair with xi_L at the required weighted scale.
```

This is stronger and cleaner than E73.017 alone.  `LOCAL-FIN` proves that the
image of the primitive module lands in a finite source module:

```text
(H_L-mu_L I)P_{A,K} subset S_{A,K}+R_72.
```

But `NAT-SRC` needs the reverse source-specific statement:

```text
S_A in (H_L-mu_L I)P_{A,K}+R_72.
```

That reverse statement is exactly `COEFF-DEF`.

## 8. Immediate finite verification target

The next computable certificate is not an SVD distance in ambient mesh norm.
It is the rank and residual of the coefficient matrix:

```text
rank B_L(A,K),
dist_coeff(s_A, Image B_L(A,K)),
support of the defect eps_A.
```

A numerical pass is meaningful only if:

```text
1. the basis is the fixed rational/Hermite basis above;
2. the residual is reported in coefficient slots, not only mesh norm;
3. endpoint slots are separated from principal source slots;
4. the xi-pairing is evaluated only after this symbolic separation.
```

This is the next finite object to build.

## 9. Update from E73.163

E73.163 canonicalizes the source basis by using only one `cauchy0` slot per
critical node.  With that non-redundant basis, the strong form of `COEFF-DEF`
above is false: the residual is large in the principal `cauchy0` slots.

Thus the statement in this note should be read as a useful attempted
strengthening, not as the surviving endpoint.  The surviving theorem is the
weaker canonical scalar form:

```text
after eliminating the rational and endpoint slots generated by the local
action, the remaining cauchy0 defect D_A satisfies

e^(alpha L)|<xi_L,D_A>| <= L^B.
```

This returns the proof obligation to `CRIT-DIV`, but with a cleaner
coefficient audit: the only load-bearing defect is the canonical critical
Cauchy atom component.
