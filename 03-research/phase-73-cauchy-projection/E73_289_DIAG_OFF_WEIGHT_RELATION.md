# E73.289 - Diagonal/off-diagonal weight relation

Date: 2026-07-14.

## 1. Purpose

E73.288 refined the endpoint to:

```text
DIAG-OFF-FREQ-DIV:
sum c^off_omega W'_omega
+sum c^diag_omega U'_omega
=O_M(L^(-M)),
```

where

```text
U'_omega=W'_omega-V'_omega/L.
```

E73.289 asks whether this cancellation comes from a simple relation between
the weights `U'` and `W'`, or from the coefficient structure.

## 2. Test

Fit the weight vector `U'` by:

```text
U' approx a W' + b conjugate(W') + c.
```

Then compute the two exact block pairings:

```text
Off  = sum c^off_omega W'_omega,
Diag = sum c^diag_omega U'_omega.
```

## 3. Result

There is no simple global weight relation:

```text
affRel ~ 0.4--0.6.
```

So `U'` is not approximately generated by `W'`, its conjugate, and a constant.

Nevertheless, for the APR coefficient pairs:

```text
Off + Diag
```

is APR-small, with relative cancellation:

```text
|Off+Diag|/(|Off|+|Diag|) ~ 1e-13--1e-15.
```

Thus the identity is not a weight-only identity.  It is a dual relation
between the coefficient pair `(c^off,c^diag)` and the two independent weight
rows `(W',U')`.

## 4. Updated theorem target

The sharp theorem is:

```text
PAIR-DUAL-FREQ-DIV:
For eta=R_wxi_L and q_a in the Cauchy plane,

<c^off_{a,w}, W'>
= -<c^diag_{a,w}, U'> + O_M(L^(-M)).
```

This is the same as `DIAG-OFF-FREQ-DIV`, but emphasizes that neither side is
small and no scalar weight simplification explains it.

## 5. Interpretation

The cancellation must come from the way `eta=R_wxi_L` solves the
Gamma-prime eigenline equation, because:

```text
1. W' and U' are not dependent as weight rows;
2. c^off and c^diag are exact but different cell sectors;
3. the pairings cancel only on the APR coefficient pair.
```

The next proof must therefore import the eigenline equation into the
frequency coefficient split, not merely analyze weights.
