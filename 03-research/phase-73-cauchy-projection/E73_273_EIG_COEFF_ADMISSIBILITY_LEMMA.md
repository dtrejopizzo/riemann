# E73.273 - EIG-COEFF admissibility lemma

Date: 2026-07-14.

## 1. Purpose

E73.272 returns the scalar branch to the principal residual

```text
APR-U4:
rho_{a,w}xi_L=q_aH_L(I-P_w)xi_L=O_M(L^-M).
```

E73.262 says the only remaining source of cancellation must be an
`EIG-COEFF` relation: a relation on the eta-packet coefficients coming from
the exact eigenline equation

```text
E_Lxi_L=0,        E_L=H_L-mu_LI.
```

This note records the admissibility lemma for such relations.  It prevents a
new circular loop: exact linear relations from the eigen-equation are
left-coborders, and left-coborders annihilate `xi_L` automatically.  They
only become useful if they leave a fixed, explicitly estimable residual.

## 2. Linear eigenline relations are left-coborders

Let `T_L` be any finite row functional built linearly from the entries of
`H_L`, `Q_w`, the CCM cells, and admissible symbolic parameters, but not from
the final scalar value `rho_{a,w}xi_L`.  If the proof of

```text
T_Lxi_L=0
```

uses only the eigenline equation `E_Lxi_L=0` by finite linear algebra, then
there is a finite primitive vector `Y_L` such that

```text
T_L=Y_L^*E_L.                                    (1)
```

Indeed, the row identities generated by left-multiplying the eigen-equation
are precisely the row module

```text
Row_E={Y^*E_L}.
```

Therefore

```text
T_Lxi_L=Y_L^*E_Lxi_L=0.                          (2)
```

This is useful algebra, but not scalar decay.

## 3. Consequence for APR-U4

Suppose one tries to prove APR-U4 by showing that the residual row is an
eigenline relation:

```text
rho_{a,w}=Y_{a,w}^*E_L.                          (3)
```

Then

```text
rho_{a,w}xi_L=0
```

would follow.  But unless `Y_{a,w}` belongs to a fixed symbolic class and (3)
is proved from explicit Gamma-prime identities, this is the unrestricted
coborder tautology from E73.005/E73.259.

More generally, if

```text
rho_{a,w}=Y_{a,w}^*E_L+r_{a,w},                  (4)
```

then APR-U4 is reduced exactly to

```text
r_{a,w}xi_L=O_M(L^-M).                           (5)
```

The coborder part disappears by (2); all content is in the residual class.

## 4. Admissible EIG-COEFF theorem

An `EIG-COEFF` proof has content only if it gives a decomposition

```text
G_{curv}=A_L^{coeff,*}E_L + R_L^{coeff},         (6)
```

or equivalently on rows

```text
rho_{a,w}=Y_{a,w}^*E_L+r_{a,w},                  (7)
```

with all of the following properties:

```text
1. Y_{a,w} is produced from a fixed symbolic module before evaluating
   rho_{a,w}xi_L;
2. r_{a,w} belongs to an explicit residual class, such as endpoint slots,
   balanced-ramp curvature slots, or a finite Loewner displacement remainder;
3. the estimate r_{a,w}xi_L=O_M(L^-M) is proved directly from the explicit
   Gamma-prime cell formula, not by using Q_wxi_L small;
4. the construction does not choose Y_{a,w} by a full pseudoinverse against
   the target row.
```

Then and only then:

```text
EIG-COEFF-COB + residual estimate
=> APR-U4
=> UNIF-ETA
=> scalar WRL.
```

## 6. Current finite identity to prove

After the admissibility lemma, the surviving identity can be stated in a
single line:

```text
Find an explicit symbolic primitive Y_{a,w} and residual r_{a,w} such that

q_aH_L(I-P_w)=Y_{a,w}^*(H_L-mu_LI)+r_{a,w},

and prove r_{a,w}xi_L=O_M(L^-M).
```

In balanced second-Abel / Loewner language, this is the same as:

```text
F_L[B_a]
=F_L[q_aQ_yR_wxi_L]
=r_{a,w}xi_L,
```

where the coborder part is invisible to the eigenline.  The residual must be
smaller by explicit Gamma-prime/Loewner cancellation, not by projection.
