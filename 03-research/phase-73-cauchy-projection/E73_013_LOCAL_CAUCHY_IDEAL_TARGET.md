# E73.013 - Local Cauchy ideal target

## 1. Purpose

E73.009--E73.012 identify the stable proof object for `IMAGE-ID`.

Raw coefficients are unstable.  A symmetric product denominator is too rigid.  The correct finite
object is the local Cauchy interpolation ideal generated by the separate off-node denominators.

## 2. Local ideal

For off-line nodes `O_L` and critical nodes `K_L`, define

```text
I_{O,K}^{DD}
= span{
   DD_L(-gamma,d)/(d^2+beta^2)^r :
   gamma in K_L,
   beta in O_L,
   r=1,2
 }.
```

This is the tested `RAT-DD_2` class.

The image ideal is

```text
J_{O,K}:=(C_E-mu I) I_{O,K}^{DD}.
```

## 3. Finite image theorem target

For every source `S_b` from E73.003, prove:

```text
S_b = J_b + R_b,
J_b in J_{O,K},
e^(Re b L)|<xi,R_b>| <= L^B,
||R_b||/||S_b|| <= L^B e^(-cL) or an already closed Phase 72 residual.
```

The last norm condition may be weakened if only the eigenline pairing is used, but the numerical
evidence supports a strong image statement:

```text
dist(S_b,J_{O,K}) about 10^-6 in tested finite windows.
```

## 4. Why separate denominators are essential

E73.012 shows that replacing the generators by a symmetric product denominator loses the local
interpolation structure.  The residue formula of E73.002 also uses separate poles at `-beta`, so the
ideal must remember each denominator independently.
