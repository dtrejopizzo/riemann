# E73.274 - APR symbolic coborder target

Date: 2026-07-14.

## 1. Purpose

E73.273 proves that `EIG-COEFF` has content only if it constructs a fixed
symbolic coborder plus an explicit residual:

```text
q_aH_L(I-P_w)=Y_{a,w}^*(H_L-mu_LI)+r_{a,w},
r_{a,w}xi_L=O_M(L^-M).
```

This note adapts the coefficient-defect architecture of E73.162 to the
principal residual `APR-U4`.  It identifies exactly what has to be built and
what cannot be recycled from the old off-line source theorem.

## 2. APR source row

Fix an admissible Cauchy plane

```text
Q=Q_w=(q_1;q_2),
P=Q^*(QQ^*)^(-1)Q,
R=I-P.
```

For row `q_a`, define the APR row

```text
rho_a=q_aH_LR.                                  (1)
```

Equivalently,

```text
rho_a=q_aH_L-A_aQ,
A=QH_LQ^*(QQ^*)^(-1).                           (2)
```

Thus `rho_a` is explicitly computable from:

```text
1. the Gamma-prime finite matrix H_L;
2. the Cauchy rows q_1,q_2;
3. the 2x2 principal action A.
```

No eigenvector is needed to define `rho_a`.

The scalar theorem is:

```text
APR-U4:
rho_axi_L=O_M(L^-M).                            (3)
```

## 3. Why E73.162 does not immediately apply

E73.162 considered a source vector `S_A` with a prescribed rational/Hermite
source module.  There the source was independent of the Cauchy-principal
projection and had a natural off-line Hermite basis.

Here the source is the APR row (1).  It contains the subtraction

```text
A_aQ,
```

where `A` itself is built from the same finite Gamma-prime matrix.  Therefore
the APR analogue of E73.162 cannot be:

```text
S_A in (H_L-mu_LI)P_{A,K}+endpoint slots.
```

It must instead be:

```text
rho_a in (H_L-mu_LI)P_{APR,w}+APR-residual slots. (4)
```

The primitive module must know about the Cauchy plane, but it must be fixed
symbolically from `q_1,q_2,D,J` and the Loewner cell formula, not chosen by
projection against `rho_a` or `xi_L`.

## 4. Candidate APR primitive module

The prior failures constrain the module:

```text
E73.242: polynomial Cauchy primitives are insufficient.
E73.243: fixed geometric rational denominators are insufficient.
E73.244: first DD-local module is insufficient as direct closure.
E73.256: quotienting by generated coborders cannot create scalar decay.
```

The smallest module not yet ruled out as a symbolic identity is the coupled
Loewner-principal module:

```text
P_{APR,w}^{CL}
= span over j=1,2 of
   q_j^*,
   D^2 q_j^*,
   D^4 q_j^*,
   D(sin(yD))[J]q_j^* integrated against the balanced second-Abel kernel,
   cos(yD)q_j^* integrated against the same kernel,
```

where the last two entries are not free continuum families.  They mean the
finite rows obtained by applying the same finite functional `F_L` that defines
APR:

```text
C_j^{cos}=F_L[cos(yD)q_j^*],
C_j^{Lo}=F_L[D(sin(yD))[J]q_j^*],
```

and then closing under the rational Cauchy denominators forced by
`q_j(n)=1/(+-w+i d_n)`.

This module is admissible only if its basis rows are written by explicit
finite formulas before testing `rho_axi_L`.

## 5. APR coefficient-defect theorem

Define the coefficient action

```text
B_{APR,L}(w): P_{APR,w}^{CL} -> S_{APR,w}^{CL}
```

by expanding

```text
(H_L-mu_LI)Y
```

for each primitive basis element `Y` into a fixed source basis
`S_{APR,w}^{CL}` generated by:

```text
1. q_aH_LR principal rows;
2. the balanced-ramp curvature endpoint slots;
3. the coupled Loewner displacement slots from [D,Lo_y]=[sin(yD),J].
```

The APR symbolic coborder theorem is:

```text
APR-COB:
there exists y_{a,w} in P_{APR,w}^{CL} such that

rho_a - y_{a,w}^*E_L = r_{a,w},
```

where `r_{a,w}` is supported only in the explicit APR residual slots and

```text
r_{a,w}xi_L=O_M(L^-M).                          (5)
```

Then `APR-U4` follows immediately.

## 6. What must be checked before this is a theorem

The module in §4 is only a target until the following finite facts are proved:

```text
APR-COB-1:
  closure: E_L P_{APR,w}^{CL} lands in S_{APR,w}^{CL} plus named endpoint
  residuals by explicit partial fractions and Loewner commutators.

APR-COB-2:
  source coverage: the APR row rho_a has source coordinates in
  S_{APR,w}^{CL} without using xi_L.

APR-COB-3:
  coefficient solvability: those source coordinates lie in Image B_{APR,L}
  modulo residual slots.

APR-COB-4:
  residual pairing: every named residual slot has xi_L-pairing O_M(L^-M).
```

E73.162 established this architecture for an off-line source, but its
canonical update E73.163 showed that the strongest coefficient-defect form was
false there.  Thus APR-COB must be verified in its own principal-residual
coordinates; it cannot be imported.

## 7. Immediate finite falsifier

Before proving symbolic partial fractions, the correct numerical falsifier is
not an ambient row norm fit.  It is:

```text
1. build the fixed APR source basis S_{APR,w}^{CL};
2. express rho_a in that basis;
3. build the coefficient action B_{APR,L};
4. solve the coefficient image problem in source coordinates;
5. report which residual slots remain;
6. only then pair the residual with xi_L.
```

If the remaining residual has a principal Cauchy slot comparable to
`rho_axi_L`, the module is insufficient.  If only endpoint/Loewner residual
slots remain and their pairing is tiny, APR-COB becomes a proof candidate.
