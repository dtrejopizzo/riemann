# E73.293 - Two-symbol Loewner normal form

Date: 2026-07-14.

## 1. Purpose

E73.292 wrote the endpoint kernel as

```text
K=qM_U-(1/(iL))q Lambda_F,
F=W^odd,
U=U^even.
```

The tempting next simplification is to absorb `M_U` as the diagonal of the
same Loewner derivative.  E73.293 checks and corrects this.  The result is a
two-symbol normal form, not a one-symbol Loewner form.

## 2. Differential identity

For the closed Gamma-prime weights,

```text
W_omega = F_L[e^(i omega y)],
V_omega = F_L[y e^(i omega y)].
```

Differentiating under the finite prime sum and the archimedean integral gives

```text
dW_omega/domega = i V_omega.                    (1)
```

Therefore, for

```text
F(omega)=W_omega-W_-omega,
```

one has

```text
F'(omega)=i(V_omega+V_-omega)=i V^even_omega.
```

The natural diagonal of the off-diagonal Loewner term in E73.292 is therefore

```text
-(1/(iL))F'(d_j) = -V^even_j/L.                 (2)
```

But

```text
U^even_j=(W_j-V_j/L)+(W_-j-V_-j/L)
        =W^even_j-V^even_j/L.                  (3)
```

Combining (2)--(3):

```text
U^even_j = W^even_j -(1/(iL))F'(d_j).           (4)
```

Thus the diagonal is not generated by `W^odd` alone.  It contains an additional
even-weight diagonal `W^even`.

## 3. Correct normal form

Let `Lambda_F^full` be the full Loewner matrix with diagonal

```text
Lambda_F^full(j,j)=F'(d_j).
```

Then

```text
K
= qM_{W^even} - (1/(iL))q Lambda_F^full.        (TS-L)
```

Equivalently, the scalar WRL endpoint is:

```text
q[ M_{W^even}-(1/(iL))Lambda_{W^odd}^{full} ]R_wxi_L
=O_M(L^(-M)).                                  (TS-DIV)
```

This is the correct analytic normal form.

## 4. Numerical falsifier for the one-symbol form

`E73_293_loewner_diagonal_probe.py` compares `U^even` with the natural
Loewner diagonal

```text
-(1/(iL)) d/domega(W_omega-W_-omega).
```

The raw relative error is order one.  Even after fitting a scalar multiple and

Loewner closure is false.

## 5. Consequence

The remaining theorem must exploit the coupling of two pieces:

```text
even diagonal:       qM_{W^even}eta,
odd Loewner term:   -(1/(iL))q Lambda_{W^odd}^{full}eta.
```

Taking either part alone is not proof-facing.  This matches the Phase 73
lesson that the scalar center is a signed coupling, not a row-norm or
single-symbol estimate.

The next analytic lemma should therefore be stated as a two-symbol directional
identity:

```text
Two-Symbol Loewner Eigenline Cancellation.
For eta=R_wxi_L,

q[ M_{W^even}-(1/(iL))Lambda_{W^odd}^{full} ]eta
=O_M(L^(-M)).
```

By E73.291, this is exactly the Schur-Commutator eigenline theorem.

## 6. Program audit

This step avoids repeating the earlier generic Loewner-commutator route:

```text
old: E72.54/E72.62 gave cell-level Loewner commutators;
new: E73.293 proves the final scalar row is not a one-symbol Loewner
     commutator, but a two-symbol even/odd coupling.
```

The distinction matters: a proof that estimates only the divided-difference
operator misses the `W^even` diagonal, and a proof that estimates only the
diagonal misses the cancellation with the odd Loewner term.
