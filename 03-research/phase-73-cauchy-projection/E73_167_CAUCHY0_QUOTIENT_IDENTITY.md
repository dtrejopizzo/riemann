# E73.167 - Cauchy0 quotient identity

Date: 2026-07-12.

## 1. Purpose

E73.165 found a rank-three projector

```text
T_L: C_L -> C_L,
```

where `C_L` is the five-dimensional canonical `cauchy0` coefficient space.
E73.167 explains this rank-three structure as a quotient:

```text
C_L / (C_L cap M_L),
```

where `M_L` is the principal coefficient image of the finite local action
`(H_L-mu_L I)` on the rational primitive module.

## 2. Definitions

Let

```text
C_L = span{DD_L(-gamma_j,d) : gamma_j in K_L, j=1,...,5}
```

in canonical coefficient coordinates.

Let

```text
M_L = Image B_L(A,K)^principal
```

where `B_L(A,K)` is the coefficient action from E73.162--E73.163 after
endpoint slots are removed.

The cauchy0 defect is the class of `Pi_A` in the quotient:

```text
delta_A = class(Pi_A) in C_L/(C_L cap M_L),
```

represented by the orthogonal projector `T_L`.

## 3. Numerical quotient test

The probe computes principal angles between `C_L` and `M_L`.  In the trusted
windows, where the local action is represented in the source basis with
small error:

```text
imageRel <= 1e-3,
```

the singular values of `Q_C^*Q_M` are:

```text
1, 0.996--1.000, about 1e-6, about 1e-8, about 1e-10.
```

Therefore:

```text
dim(C_L cap M_L)=2,
dim C_L/(C_L cap M_L)=3.
```

This explains the rank-three projector from E73.165.

## 4. Untrusted large-window rows

For larger diagnostic windows, the coefficient extraction has:

```text
imageRel = 3e-2 to 2.6e-1.
```

Those rows are not reliable for this coefficient-intersection test.  They
should not be used to infer the exact dimension of `C_L cap M_L`.

The rank-three transfer test of E73.165 remains numerically stable there,
but the quotient explanation should be certified first in better-conditioned
coordinates or by symbolic partial fractions.

## 5. Finite identity

The current finite identity can now be stated without reference to a
least-squares source residual:

```text
There is a two-dimensional subspace I_L := C_L cap M_L of cauchy0 sources
which are generated by the finite rational local action.  Only the quotient

Q_L := C_L/I_L

is seen by the canonical cauchy0 defect.
```

Let

```text
P_{Q,L}: C_L -> Q_L
```

be the quotient projector.  Then the remaining theorem is:

```text
QUOT-CRIT-DIV:
e^(alpha L)
| < P_{Q,L}Pi_A, G_K > |
<= L^B.
```

Here

```text
G_K(gamma)
=(1-exp(i gamma L))sum_n xi_L(n)/(-gamma-d_n).
```

## 7. What remains

