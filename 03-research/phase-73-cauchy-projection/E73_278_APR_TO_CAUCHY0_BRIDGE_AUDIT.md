# E73.278 - APR-to-cauchy0 bridge audit

Date: 2026-07-14.

## 1. Purpose

E73.277 identified the only named non-tautological residual slot currently
available: the canonical three-dimensional `cauchy0` quotient of
E73.163--E73.167.  The immediate possible bridge was:

```text
APR-to-cauchy0:
the APR principal residual rho=qH(I-P_w) descends to the same cauchy0
quotient, so APR-U4 can be reduced to QUOT-CRIT-DIV.
```

E73.278 tests this bridge directly.

## 2. Test

For each critical Cauchy plane `Q_w`, form the APR row

```text
rho_j(w)=q_jH_L(I-P_w).
```

Fit `rho_j(w)` into the canonical source basis of E73.163:

```text
cauchy0_gamma(d)=DD_L(-gamma,d),
rat_{gamma,beta,r}(d)=DD_L(-gamma,d)/(d^2+beta^2)^r,
endpoint_{gamma,q}(d)=d^(2q)DD_L(-gamma,d).
```

Then project only the `cauchy0` coefficient part to the quotient

```text
C_L/(C_L cap M_L)
```

identified in E73.167, and compare scalar pairings with `xi_L`:

```text
center      = rho_j(w)xi_L,
quot_center = cauchy0_quotient(rho_j(w))xi_L,
rem_center  = (rho_j(w)-cauchy0_quotient(rho_j(w)))xi_L.
```

If APR-to-cauchy0 were the correct bridge, the quotient component would carry
the center and the remaining scalar would be negligible.

## 3. Result

In trusted rows, where both the canonical fit and the coefficient image are
well-conditioned enough to read the test, the bridge fails:

```text
quot/center ranges from about 1e-3 to 1e5,
rem/center is often order 1 or much larger,
```

and for the first critical node the quotient and remainder are frequently much
larger than the actual APR center.  That means the small APR scalar is not the
same cancellation as the old `cauchy0` quotient scalar.

Large-window rows with `imageRel` around `5e-2` are not used as proof-quality
evidence for the quotient dimension, but they do not repair the pattern.

## 4. Meaning

The clean fork from E73.277 is now resolved:

```text
APR-U4 does not reduce, in the tested canonical coordinates, to the old
three-dimensional cauchy0 quotient.
```

This does not disprove APR-U4.  It says the old residual slot is not the
missing slot for the APR principal residual.

The observed pattern is also structurally important: `quot_center` and
`rem_center` can both be much larger than `center`, so APR-U4 depends on
cancellation between canonical coefficient sectors.  A proof that throws away
the non-cauchy0 sectors or takes absolute values of them will miss the
mechanism.

## 5. Updated obstruction

After E73.278, the only honest statement is:

```text
APR-U4 remains a coupled canonical-sector cancellation:

rho_j(w)xi_L
= [cauchy0 quotient contribution]
 + [rational/endpoint/generated-sector contribution]
 + [coordinate fitting residual contribution],
```

where the first two terms are individually not small enough.

Therefore the next residual slot cannot be a pure `cauchy0` quotient.  It must
be a coupled quotient retaining the signed interaction between:

```text
1. cauchy0 critical atoms;
2. rational beta-dependent atoms;
3. endpoint atoms generated by the local action.
```

## 6. Next target

The next finite theorem should be formulated as a coupled-sector quotient:

```text
COUPLED-APR-QUOT:
Let S_L be the canonical source space of E73.163 and M_L the principal image
of the finite local action.  Let rho_j(w) be the APR row.  Prove that the class

    [rho_j(w)] in S_L/M_L

pairs with xi_L by O_M(L^(-M)), without projecting to cauchy0 alone.
```

This avoids the left-coborder tautology because the quotient is a fixed
canonical coefficient quotient, not an ambient rowspace projection; and it
avoids the cauchy0 false simplification because it keeps the coupled sectors
that numerically cancel.
