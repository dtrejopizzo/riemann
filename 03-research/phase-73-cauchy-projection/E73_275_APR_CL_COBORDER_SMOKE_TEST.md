# E73.275 - APR coupled-Loewner coborder smoke test

Date: 2026-07-14.

## 1. Purpose

E73.274 proposes a coupled Loewner-principal symbolic coborder for APR-U4:

```text
rho_a=q_aH_L(I-P_w)
=Y_{a,w}^*(H_L-mu_LI)+r_{a,w},
r_{a,w}xi_L=O_M(L^-M).
```

This note tests the first finite proxy for such a module.  It is a falsifier,
not a proof: the rows are generated before pairing with `xi_L`, and the scalar
pairing is checked only afterward.

## 2. Tested modules

For the Cauchy plane `Q_w=(q_1;q_2)`, the script compares:

```text
poly:
  span{D^kq_j^*: 0<=k<=4, j=1,2}.

dd:
  the first DD-local module from E73.244.

cl:
  q_j^*, D^2q_j^*, D^4q_j^*,
  H_Lq_j^*, H_L^Aq_j^*, Prime_Lq_j^*,
  and D^2 times these action vectors.

dd+cl:
  combined DD-local and CL-action proxy.
```

For each module `P`, it forms

```text
M_P={Y^*E_L: Y in P},       E_L=H_L-mu_LI,
```

projects `rho_a` onto `M_P`, and measures:

```text
res=rho_a-Proj_{M_P}rho_a,
res xi_L,
||res||,
||Proj rho_a||,
||res Q_w^*||.
```

## 3. Result

The stable windows show:

```text
res xi_L ~= rho_a xi_L,
```

as required by exact left-coborder invariance.  The CL-action proxy improves
some row geometry and Cauchy-plane leakage compared with DD-local, but it does
not make the residual row small enough to be interpreted as a named endpoint
or Loewner residual slot.

In particular, the residual norm remains many powers of `L` larger than the
scalar center.  Therefore this proxy does not supply the APR-COB residual
estimate.

## 4. Interpretation

The failure is useful:

```text
1. Adding H_Lq_j^*, H_L^Aq_j^*, and Prime_Lq_j^* as primitive columns is not
   the missing symbolic coborder.
2. The balanced second-Abel functional has to enter at the source-coordinate
   level, not merely through action-generated primitive vectors.
3. The scalar cancellation remains APR-U4 itself, not a consequence of row
   geometry.
```

This sharpens E73.274.  The next falsifier/proof attempt should build the
source basis from the coefficient curvature slots

```text
U_omega^bal, Z_omega^bal
```

and test coefficient-image solvability there, rather than projecting mesh rows
generated by action vectors.

## 5. Status

```text
tested: poly, DD-local, CL-action, and DD+CL primitive proxies;
rejected: CL-action proxy as direct APR-COB closure;
survives: source-coordinate balanced second-Abel APR-COB target;
open: build the coefficient/source basis for curvature slots and test image
      solvability before pairing with xi_L.
```
