# E73.015 - Local ideal action rules

## 1. Purpose

E73.014 shows that the local action closes numerically after adding endpoint/polynomial generators.
This note states the symbolic rules that must be proved from the coupled finite operator.

## 2. Generators

Let

```text
DD_gamma(d):=DD_L(-gamma,d).
```

The primitive local ideal is generated by

```text
Y_{gamma,beta,r}(d)=DD_gamma(d)/(d^2+beta^2)^r,    r=0,1,2,
P_gamma(d)=d^2 DD_gamma(d).
```

The source ideal is generated by

```text
DD_gamma(d)/(d^2+beta^2)^r,        r=0,1,2,3,
d^2 DD_gamma(d),
d^4 DD_gamma(d).
```

## 3. Formal action rule

Original strict target:

```text
(C_E-mu I)Y_{gamma,beta,r}
= sum c_{r,j} DD_gamma(d)/(d^2+beta^2)^j
 + polynomial endpoint terms
 + Phase72Residual,
```

with `j<=r+1`.  Similarly,

```text
(C_E-mu I)P_gamma
= polynomial endpoint terms up to d^4 DD_gamma
 + lower rational terms
 + Phase72Residual.
```

The desired feature was triangularity:

```text
denominator order increases by at most one.
```

This is exactly what the E73.014 source ideal includes.

E73.016 corrects this statement.  Strict triangularity is numerically false as an exact
finite rule, mainly in the lower rational blocks.  The surviving target is finite-width:

```text
(C_E-mu I)Y_{gamma,beta,r}
= sum_{j=0}^3 c_j DD_gamma(d)/(d^2+beta^2)^j
 + endpoint terms d^2 DD_gamma(d), d^4 DD_gamma(d)
 + Phase72Residual.
```

Thus the load-bearing replacement is `LOCAL-FIN`, not strict `LOCAL-RULE`.

## 4. Why triangularity is plausible

The CCM cell kernels are built from finite Cauchy/Loewner divided differences in the pole mesh.
Multiplication by the mesh variable and one Loewner commutator can only:

```text
1. preserve the critical divided-difference factor DD_gamma;
2. raise the local pole order at d^2=-beta^2 by one;
3. generate endpoint polynomial terms from the finite interval boundary.
```

The prime/archimedean pieces must be kept coupled; triangularity is a statement about
`W02-WR-Prime`, not about the three terms separately.

## 5. Proof target

The next proof should establish:

```text
LOCAL-RULE:
For every generator of the primitive ideal, the coupled action of C_E-mu has the triangular local
form above, with residuals in the already controlled Phase 72 classes.
```

Then E73.014 becomes a finite algebraic certificate rather than a numerical subspace observation.
