# E73.163 - Canonical coefficient-defect audit

Date: 2026-07-12.

## 1. Purpose

E73.162 proposed a coefficient-level version of the source coborder:

```text
s_A - B_L(A,K)y_A = eps_A,
```

where `B_L(A,K)` is the action of `(H_L-mu_L I)` from a fixed rational
primitive module into a fixed rational source module.

The E73.162 smoke test looked promising, but it still used a redundant
source basis: the `r=0` Cauchy column

```text
DD_gamma(d)
```

was repeated once for every off-line beta.  This can make coefficient
residuals artificially small because the same function is represented in
many coordinate slots.

E73.163 removes this redundancy.

## 2. Canonical basis

For each critical node `gamma`, use one and only one principal Cauchy column:

```text
cauchy0_gamma(d)=DD_L(-gamma,d).
```

For each off-line beta, use only the genuinely beta-dependent columns:

```text
rat_{gamma,beta,r}(d)=DD_L(-gamma,d)/(d^2+beta^2)^r,
r=1,2,3.
```

Endpoint slots are kept separate:

```text
endpoint_{gamma,q}(d)=d^(2q)DD_L(-gamma,d),
q=1,2.
```

In this canonical basis the source coefficients are exact, not extracted by
least squares:

```text
s_A(cauchy0_gamma)=Pi_conf(A,i gamma),
s_A(other slots)=0.
```

Thus this is a sharper test of `COEFF-DEF`.

## 3. Result

The canonical coefficient test gives:

```text
principalRel = 1e-1 to 7e-1,
cauchy0Rel   = 1e-1 to 7e-1,
endpointRel  = 4e-4 to 6e-2,
ratRel       = about 1e-3 to 4e-3.
```

The large residual is concentrated in the `cauchy0` principal slots.
Endpoint and higher rational slots are comparatively small.

At the same time, the scalar weighted residual remains tiny:

```text
expPair = 1e-13 to 1e-8
```

in the tested rows.

## 4. Meaning

This falsifies the strong form of `COEFF-DEF` from E73.162:

```text
S_A in (H_L-mu_L I)P_{A,K} + endpoint slots
```

in canonical coefficient coordinates.

The apparent small coefficient defect in E73.162 was a coordinate artifact
caused by redundant `r=0` columns.  Once the principal Cauchy coordinate is
unique, the source is not absorbed by the coefficient image modulo endpoint
slots.

## 5. What survives

The scalar channel is still small.  Therefore the true theorem is not a
coefficient-image theorem for the whole principal source vector.

The surviving statement is:

```text
CANON-SCALAR:
The canonical principal defect

delta_A := s_A - B_L(A,K)y_A

may have large cauchy0 coefficients, but the mesh vector represented by
delta_A is xi_L-small after the natural off-line weight:

e^(alpha L)|<xi_L, S(delta_A)>| <= L^B.
```

This is weaker than `COEFF-DEF`, but it is not the old ambient projection
tautology because:

```text
1. y_A is chosen by matching the canonical rational action coefficients;
2. delta_A is explicitly localized to cauchy0 principal slots;
3. the remaining proof target is a finite scalar statement about those
   canonical cauchy0 coefficients.
```

## 6. Updated endpoint

The Phase 73 endpoint sharpens from:

```text
source lies in coefficient image modulo endpoint
```

to:

```text
canonical cauchy0 defect is xi_L-small.
```

Equivalently, after eliminating all rational and endpoint slots that are
genuinely generated by the local action, the remaining obstruction is a
finite source supported on the critical Cauchy atoms:

```text
D_A(d)=sum_{gamma in K_L} delta_A(gamma) DD_L(-gamma,d),
```

and the required theorem is:

```text
e^(alpha L)|<xi_L,D_A>| <= L^B.
```

This is essentially `CRIT-DIV`, now with a canonical coefficient reduction
showing that all non-Cauchy principal clutter has been stripped away.
