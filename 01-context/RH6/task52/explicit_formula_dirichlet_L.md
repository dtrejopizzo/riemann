
# Guinand-Weil Explicit Formula for Dirichlet L-functions
# ======================================================= ## Conventions Let h: R -> C be an even Schwartz function (or more generally, analytic in |Im s| < 1/2 + epsilon with h(s)(1+|s|)^{1+delta} bounded). Define the Fourier transform: g(u) = (1/(2*pi)) * integral_{-infty}^{infty} h(t) e^{-iut} dt Inverse: h(t) = integral_{-infty}^{infty} g(u) e^{iut} du ## Formula for the Riemann Zeta Function Sum_{rho: zeta(rho)=0} h(gamma_rho) = h(i/2) + h(-i/2) - g(0) * log(pi) + (1/(2*pi)) * integral_{-infty}^{infty} h(t) * Re psi(1/4 + it/2) dt - 2 * Sum_{n >= 1} Lambda(n) * n^{-1/2} * g(log n) where:
- gamma_rho runs over ALL imaginary parts of non-trivial zeros (both positive and negative)
- Under RH, rho = 1/2 + i*gamma, so h(gamma_rho) = h(gamma)
- Without RH, h(gamma_rho) should be read as h((rho - 1/2)/i)
- psi = Gamma'/Gamma is the digamma function
- Lambda(n) is the von Mangoldt function ## Formula for L(s, chi), chi primitive non-principal mod q ### Statement Let chi be a primitive non-principal Dirichlet character modulo q >= 2.
Define: a_chi = 0 if chi(-1) = +1 (chi is EVEN) a_chi = 1 if chi(-1) = -1 (chi is ODD) Then: Sum_{rho: L(rho,chi)=0} h((rho - 1/2)/i) = g(0) * log(q/pi) + (1/(2*pi)) * integral_{-infty}^{infty} h(t) * Re psi(1/4 + a_chi/2 + it/2) dt - (1/(2*pi)) * Sum_{n >= 2} (Lambda(n)/sqrt(n)) * {chi(n)*h_hat(log(n)/(2*pi)) + conj(chi(n))*h_hat(-log(n)/(2*pi))} where h_hat(xi) = integral h(x) e^{-2*pi*i*x*xi} dx is the Fourier transform in the frequency convention. Converting to the g convention (h_hat(xi) = 2*pi*g(2*pi*xi)): Sum_{rho: L(rho,chi)=0} h((rho - 1/2)/i) = g(0) * log(q/pi) + (1/(2*pi)) * integral_{-infty}^{infty} h(t) * Re psi(1/4 + a_chi/2 + it/2) dt - Sum_{n >= 2} (Lambda(n)/sqrt(n)) * {chi(n)*g(log n) + conj(chi(n))*g(-log n)} ### Simplification for h even (hence g even and real-valued) If h is even (as in the user's setup), then g(-u) = g(u), and: ***********************************************************************

*********************************************************************** ### Detailed breakdown by parity of chi: CASE 1: chi EVEN (chi(-1) = +1, a_chi = 0) Gamma factor in completed L-function: pi^{-s/2} Gamma(s/2) Formula: Sum_{rho} h((rho-1/2)/i) = g(0)*log(q/pi) + (1/(2*pi)) * int h(t) Re psi(1/4 + it/2) dt - 2 * Sum_{n>=2} Lambda(n) Re(chi(n)) n^{-1/2} g(log n) NOTE: The digamma argument psi(1/4 + it/2) is the SAME as for zeta. CASE 2: chi ODD (chi(-1) = -1, a_chi = 1) Gamma factor in completed L-function: pi^{-(s+1)/2} Gamma((s+1)/2) Formula: Sum_{rho} h((rho-1/2)/i) = g(0)*log(q/pi) + (1/(2*pi)) * int h(t) Re psi(3/4 + it/2) dt - 2 * Sum_{n>=2} Lambda(n) Re(chi(n)) n^{-1/2} g(log n) NOTE: The digamma argument shifts from psi(1/4 + it/2) to psi(3/4 + it/2). ## Answers to the user's specific questions: (1) h(i/2) + h(-i/2) terms: NO, these do NOT appear for non-principal chi. They come from the poles of zeta at s=0 and s=1 (which correspond to r(pi)=1 for zeta). Since L(s,chi) is entire for non-principal chi, r(pi)=0 and these terms vanish. (2) The conductor/pi term: g(0) * log(q/pi) This replaces -g(0)*log(pi) from the zeta formula. The full conductor appears as q (the modulus), not q^2 or anything else, because the conductor of a degree-1 L-function is just q. (3) The archimedean digamma integral: Coefficient: 1/(2*pi) [SAME as for zeta] chi EVEN: (1/(2*pi)) * int_{-infty}^{infty} h(t) Re psi(1/4 + it/2) dt chi ODD: (1/(2*pi)) * int_{-infty}^{infty} h(t) Re psi(3/4 + it/2) dt (4) The prime sum: -2 * Sum_{n >= 2} Lambda(n) * Re(chi(n)) * n^{-1/2} * g(log n) The factor in front is -2 (same as zeta). The character enters as Re(chi(n)), NOT chi(n) directly or 2*Re(chi(n)). For real characters (quadratic), Re(chi(n)) = chi(n), and this reduces to: -2 * Sum_{n >= 2} Lambda(n) * chi(n) * n^{-1/2} * g(log n) (5) The LHS sum: The sum runs over ALL non-trivial zeros of L(s,chi), parametrized as h((rho-1/2)/i) where rho ranges over zeros with 0 < Re(rho) < 1. Under GRH (rho = 1/2 + i*gamma), this becomes Sum_{gamma in R} h(gamma), where gamma runs over BOTH positive and negative values. Each gamma and -gamma are counted separately (they need not be paired since L(s,chi) does not have the symmetry zeta(s) = conj(zeta(conj(s))) unless chi is real). For real chi: if rho is a zero, so is conj(rho) = 1-conj(rho), and under GRH the zeros come in conjugate pairs gamma, -gamma. For complex chi: the zeros of L(s,chi) and L(s,conj(chi)) are related but the zeros of L(s,chi) alone need not be symmetric about gamma=0. ## Sources 1.
