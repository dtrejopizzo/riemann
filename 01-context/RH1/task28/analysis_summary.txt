 RESEARCH HYPOTHESIS:
The GEV shape parameter ξ for resonance metrics, when viewed as a function of
height t, will exhibit a qualitatively different pattern for the validated L_DH
compared to periodic multiplicative functions (ζ, L(s,χ₄)). Specifically, ξ(t) for L_DH will show a consistent monotonic trend (e.g., decreasing with t), while
for ζ and L(s,χ₄) it will be stable or non-monotonic. RESULT:
The hypothesis is REJECTED. The validated L_DH does NOT show a qualitatively
different trend pattern compared to multiplicative functions. KEY FINDINGS: 1. TREND DIRECTION (Contradicts Hypothesis): ALL three functions show POSITIVE trends (ξ increasing with t), not the hypothesized decreasing trend for L_DH: • ζ: slope = 9.74×10⁻⁶ per unit t (p=0.042, significant) • L(χ₄): slope = 5.29×10⁻⁶ per unit t (p=0.251, not significant) • L_DH: slope = 5.88×10⁻⁶ per unit t (p=0.195, not significant) The slopes are NOT significantly different from each other (all pairwise p-values > 0.4). This indicates similar temporal evolution of extreme value statistics across all three function classes. 2. MEAN LEVEL (Distinguishing Feature): While the trends are similar, L_DH has a significantly HIGHER mean ξ compared to multiplicative functions: • L_DH: ξ̄ = -0.0538 ± 0.0717 • L(χ₄): ξ̄ = -0.1116 ± 0.0720 • ζ: ξ̄ = -0.1678 ± 0.0819 Mann-Whitney U test: p = 0.0165 (significant at α=0.05) The higher (less negative) ξ for L_DH suggests it has a HEAVIER tail in the distribution of resonance maxima. This is consistent with the presence of off-line zeros. 3. VARIANCE: All three functions have similar variance in ξ across windows: • ζ: Var(ξ) = 0.0067 • L(χ₄): Var(ξ) = 0.0052 • L_DH: Var(ξ) = 0.0051 4. VALIDATION OF IMPLEMENTATION: The validated L_DH implementation was confirmed correct: • CAS = 3.66 ≈ 3.6 (expected for validated version with periodic coefficients) • Coefficients are real and period-5: {1, 0.249, -0.249, -1, 0} • This matches the "Validated Historical L_DH" (version #3) from the dataset description 5. MONOTONICITY: None of the functions show consistent monotonic behavior: • ζ: 5/9 windows show increase, 4/9 show decrease • L(χ₄): 4/9 windows show increase, 5/9 show decrease • L_DH: 5/9 windows show increase, 4/9 show decrease INTERPRETATION: The validated L_DH function (with real periodic coefficients and known off-line
zeros) does NOT exhibit the hypothesized decreasing ξ(t) profile. Instead, it
shows a positive trend similar to multiplicative functions. The key distinguishing
feature is the LEVEL (mean value) of ξ, not the TREND. This suggests that:
1. The non-stationarity pattern (trend direction) of extreme value statistics is NOT a robust signature for distinguishing L-functions with off-line zeros from those satisfying RH/GRH. 2. The MEAN value of ξ (tail heaviness) may be a more reliable distinguishing feature than the temporal trend pattern. 3. The previous finding (r75) that the pattern differs between function classes is not confirmed with the validated L_DH implementation and the current analysis parameters (N=10⁵, 10 windows). 4. All three functions show increasing ξ with height t, which may reflect common behavior of Dirichlet polynomials at increasing heights rather than structural differences related to zeros. METHODOLOGICAL NOTES:
• N = 100,000 (10⁵ truncation)
• t range: [1000, 20000], divided into 10 windows of width 1900
• Block size: 50 for extracting maxima (~69 maxima per window)
• GEV fitting: Maximum likelihood estimation with bootstrap CI (1000 iterations)
• Standard sampling resolution: Δt = 2π/log(N) ≈ 0.546
