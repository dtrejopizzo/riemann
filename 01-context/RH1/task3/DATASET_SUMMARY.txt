MULTIPLICATIVE RESONANCE PROGRAM - DATASET SUMMARY
================================================================================ DATASET DESCRIPTION
-------------------
This computational dataset contains computed values of the partial sum: D_F(t; N) = Σ_{n≤N} a_n(F) / n^(1/2 + it) for different function classes F, as specified in the Multiplicative Resonance Program research document (research-program-v3.pdf, Section 3.4). The dataset is designed to test the hypothesis that multiplicative arithmetic structure imposes decorrelation on prime contributions, making coherent phase alignment (which produces off-line zeros) structurally impossible. FUNCTIONS COMPUTED
------------------
1. Riemann zeta function (ζ) - Coefficients: a_n = 1 (multiplicative) - Has Euler product: YES - RH status: Expected to hold 2. L(s, χ₄) - Dirichlet L-function with real character mod 5 - Coefficients: χ₄(1)=1, χ₄(2)=-1, χ₄(3)=-1, χ₄(4)=1, χ₄(0)=0 - Has Euler product: YES - GRH status: Expected to hold 3. Random multiplicative function (f_rand, seed=42) - Coefficients: a_p = ±1 uniform i.i.d. at primes, extended multiplicatively - Has Euler product: NO (but is multiplicative) - RH status: Unknown - tests if multiplicativity alone suppresses resonance 4. Davenport-Heilbronn function (L_DH) - Coefficients: a_n = (1-iκ)/2 χ(n) + (1+iκ)/2 χ̄(n), κ ≈ 0.3717 - Has Euler product: NO - RH status: FAILS - proven to have zeros off the critical line - Known off-line zeros: * σ = 0.8085, t ≈ 85.70 ← target zero for scaling analysis * σ = 0.6508, t ≈ 114.16 * σ = 0.5744, t ≈ 166.48 * σ = 0.7243, t ≈ 176.70 5. Perturbed Davenport-Heilbronn function (L_DH^(ε), ε=0.01) - Same as L_DH but with κ' = κ + 0.01 - Tests robustness beyond single counterexample COMPUTATIONAL PARAMETERS
------------------------
• Truncation lengths N: {10⁴, 10⁵, 10⁶}
• Height range T: [80, 92] - covers known zero at t ≈ 85.7
• Grid spacing: Δt = 2π / log(N) (Nyquist-like resolution)
• Numerical precision: float64 (np.complex128)
• Summation method: Kahan compensated summation
• Target relative error: < 10⁻¹² RESEARCH QUESTION
-----------------
Does tracking a specific peak location as it shifts with N reveal a growth exponent closer to the theoretical value (~0.31) than the global maximum method (~0.00018)? Theoretical prediction: For L_DH near the off-line zero at σ = 0.8085, t ≈ 85.7: M(N) ~ N^α where α = σ - 1/2 = 0.3085 Observed results (N ∈ [10⁴, 10⁶]): • Global maximum method: α ≈ 0.0034 • Tracked peak method: α ≈ -0.083 Conclusion: The observed exponents are much smaller than the theoretical prediction. This suggests that pre-asymptotic effects dominate in the range N ≤ 10⁶. Much larger N values or refined peak-tracking methods are needed to observe the asymptotic α ≈ 0.31 behavior predicted by the known off-line zero. KEY FINDINGS
------------
1. Peak location in |D_DH(t;N)| occurs at t ≈ 84.2, not at the known zero location t ≈ 85.7, suggesting complex peak structure with multiple local maxima. 2. Peak magnitude grows slowly from M ≈ 4.15 at N=10⁴ to M ≈ 4.22 at N=10⁶, corresponding to growth exponent α ≈ 0.0034. 3. The resonance score RS(f_rand) = 3.65 is notably high at N=10⁴, with kurtosis K = 10.5, suggesting that random multiplicative functions can exhibit extreme fluctuations. 4. Clear separation observed: RS(L_DH) = 2.16 > RS(ζ) = 1.77 at N=10⁴, consistent with the validation criteria in Section 2.4 of the research program. FILES GENERATED
