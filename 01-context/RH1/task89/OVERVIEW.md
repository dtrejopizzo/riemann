</conclusion> <methods>
The analysis implemented the computational framework specified in the research program. For each of five function classes (Riemann ζ, L(s, χ₄), f_rand with seed=42, L_DH, and L_DH with ε=0.01 perturbation), coefficient generators were implemented following the precise specifications: ζ has a_n=1; χ₄ is the real character mod 4 (χ₄(n)=0 for n even, +1 for n≡1 mod 4, -1 for n≡3 mod 4); f_rand assigns a_p=±1 uniformly at random to primes and extends multiplicatively; L_DH uses the Davenport-Heilbronn coefficients with κ=(√5-1)/(2√(5(√5-1))) and the complex character mod 5 (χ(1)=1, χ(2)=i, χ(3)=-i, χ(4)=-1). The partial sums D_F(t; N) = Σ_{n≤N} a_n/n^(1/2+it) were computed using Kahan compensated summation for numerical stability across three scales N ∈ {10⁴, 10⁵, 10⁶}. The height range T ∈ [100, 200] was sampled with grid spacing Δt = 2π/log(N), yielding 147-220 t-values per scale depending on N. This range contains three known L_DH off-line zeros at t ≈ 114.2, 166.5, and 176.7, and falls within the validated numerical precision bounds (relative error < 10⁻¹² for t < 500). Cross-scale coherence was computed as X_F(t; N₁, N₂) = D_F(t;N₁)·conj(D_F(t;N₂))/√(V_F(N₁)·V_F(N₂)) where V_F(N) is the mean square of D_F(t;N). For each scale pair (10⁴, 10⁵) and (10⁵, 10⁶), the D_F values were interpolated to a common t-grid using linear interpolation of real and imaginary parts separately. The magnitude |X_F(t)| was analyzed through summary statistics (mean, standard deviation, maximum, percentiles) and visualized using boxplots and time-series profiles. All computations used NumPy 1.x with standard precision (float64 complex128).
