## Overview The hypothesis that "the refined resonance metric R_comp will exhibit clear power-law growth with N at L_DH resonance locations" is **REFUTED** by the data. ## Key Findings **R_comp scaling behavior:**
- **L_DH** (at t=145.96, location of maximum R_comp): Shows **power-law DECAY** with exponent α = -0.533 (95% CI: well-constrained by R² = 0.9998). R_comp decreases from 0.0587 at N=10⁴ to 0.0050 at N=10⁶, a factor of 0.085.
- **ζ** (at t=108.72, location of maximum |D|): Shows **essentially flat behavior** with α = 0.0014 (effectively zero). R_comp remains nearly constant at ~0.0047-0.0049 across all N values.
- **L_DH** (at t=114.16, known off-line zero): Also shows strong decay, with R_comp decreasing from 0.0093 to 0.0016. **Model selection results:**
- **L_DH**: Power-law model is **strongly preferred** over log-correlated model (ΔAIC = -7.16, ΔBIC = -7.16), indicating decisive evidence for power-law scaling.
- **ζ**: Neither model is preferred (ΔAIC = -0.008), consistent with flat/constant behavior. **Distinction between functions:**
The R_comp metric DOES distinguish L_DH from ζ, but in the **opposite direction** from the hypothesis:
- L_DH shows strong power-law N-dependence (decay)
- ζ shows minimal N-dependence (constant) ## Interpretation The R_comp metric, defined as the mean resultant length of composite squarefree terms (|Σ terms| / Σ|terms|), is a **normalized coherence measure** bounded between 0 and 1. As N increases and more terms are added to the sum, phase decorrelation naturally causes this normalized metric to decrease. The key distinction is: - **L_DH**: Strong systematic decrease following power-law R_comp ∝ N^(-0.533)
- **ζ**: Minimal decrease, essentially independent of N This indicates that composite squarefree terms in L_DH become increasingly decorrelated as more are included, while those in ζ maintain roughly constant phase coherence regardless of truncation length. ## Conclusion on Hypothesis The original hypothesis expected power-law **GROWTH** (α > 0) for L_DH based on the idea that off-line zeros would create persistent, growing resonance signatures. The observed power-law **DECAY** (α < 0) suggests that: 1. R_comp as a normalized metric naturally decreases with sample size
2. The off-line zero does NOT create growing coherence in the composite squarefree component
3. The metric may still be useful for distinguishing function classes (power-law vs. flat N-dependence) but not as a "resonance amplification" signature The hypothesis that R_comp would reveal power-law growth as a signature of off-line zeros is **not supported**. However, the strong power-law structure in L_DH versus flat behavior in ζ provides a clear and statistically significant distinction between the two function classes. --- ## Discretionary Analytical Decisions - **Selected L_DH resonance location at t=145.96** (where R_comp is maximized at N=10⁵) rather than solely using t=114.16 (known off-line zero location). This was done after observing that t=114.16 did not show the hypothesized behavior, and to test whether the effect might be location-dependent.
- **Selected ζ peak location at t=108.72** based on maximum |D(t;N)| in range [100,150] with N=10⁵, providing a fair comparison of "peak" locations for both functions.
- **Used AIC and BIC for model selection** with threshold |ΔAIC| < 2 for "comparable models" and |ΔAIC| > 2 for clear preference, following standard information-theoretic criteria.
- **Fitted power-law model R_comp = c·N^α** allowing negative α (decay) when data showed decreasing trend, rather than forcing α > 0 as hypothesized.
- **Used N values [10⁴, 5×10⁴, 10⁵, 5×10⁵, 10⁶]** as specified in research objective, despite computational cost at N=10⁶.
- **Computed R_comp only for composite squarefree terms** (squarefree numbers that are neither 1 nor prime), following the dataset description definition.
- **Implemented L_DH coefficients using character mod 5 of order 4** with κ = (√5-1)/(2√(5(√5-1))), following the precise construction in research-program-v3.pdf.
- **Set significance threshold α = 0.05** implicitly for model comparison using AIC/BIC criteria.
