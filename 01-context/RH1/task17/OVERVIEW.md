</conclusion> <methods>
For six function classes (ζ, L(s,χ₄) mod 5, f_rand, L_DH, L(s,χ_4) mod 4, f_fully_rand), I implemented coefficient generation with proper multiplicative structure for multiplicative functions and independent random values for non-multiplicative controls. Partial sums D(t; N=10⁵) were computed over t ∈ [1000, 10000] with 9001 sampling points using the formula D_F(t;N) = Σ_{n≤N} a_n / n^(1/2 + it). For each function, the top 50 magnitude peaks were identified using scipy.signal.find_peaks. At each of 300 peak locations, I computed M_coh = |ΣS_k| / √Σ|S_k|² (where S_k partitions terms by ω(n)) and R_comp = |Σ(a_n/n^(1/2+it))| / Σ|a_n/n^(1/2+it)| for composite squarefree terms (ω(n)≥2). Statistical analysis employed one-way ANOVA for individual metrics, MANOVA using statsmodels for multivariate testing, and pairwise t-tests with Bonferroni-adjusted significance threshold (p<0.001). Linear Discriminant Analysis assessed overall separability. Sparsity was quantified as the fraction of zero coefficients, with Pearson and Spearman correlations computed against mean R_comp values.
