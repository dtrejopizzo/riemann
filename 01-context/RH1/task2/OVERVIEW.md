- Compute FFT on first 10,000 non-zero coefficients
- Extract power spectrum: |FFT|²
- Exclude DC component (zero frequency)
- Calculate ratio of maximum peak to mean power
- Apply log₁₀ transformation for scale **Interpretation:**
- High CAS (>2.5): Strong frequency peaks indicate periodic structure
- Low CAS (<1.5): Flat spectrum indicates random/pseudo-random behavior
- The metric is robust to coefficient sparsity (handles zeros appropriately) ## Key Scientific Insight The Davenport-Heilbronn function L_DH, despite being deterministically defined via the Möbius and Liouville functions, exhibits spectral properties indistinguishable from truly random functions. Its CAS score (0.9566) is virtually identical to f_rand (1.0071) and f_fully_rand (1.0585), confirming its pseudo-random nature at the coefficient level. This validates the metric's ability to capture functional behavior beyond simple determinism. ## Discretionary Analytical Decisions - **CAS metric choice:** Selected FFT-based periodicity score over lag-1 autocorrelation or multi-lag methods after testing three approaches. The FFT approach showed superior separation (clear bimodal distribution) compared to lag-based methods which couldn't distinguish L_DH from periodic functions. - **FFT window length:** Used first 10,000 non-zero coefficients for FFT analysis (balance between frequency resolution and computational efficiency). - **CAS transformation:** Applied log₁₀ transformation to FFT periodicity ratio for better scale separation and to match the scale of M_coh and R_comp features. - **Classification algorithms:** Tested both LDA and SVM (RBF kernel) as requested; LDA achieved perfect separation due to near-linear separability in 3D space. - **Cross-validation scheme:** Used 5-fold stratified cross-validation to ensure balanced class representation (50 samples per class, 300 total). - **Feature standardization:** Applied StandardScaler to all features before classification to ensure equal contribution across different scales. - **Character function definitions:** Verified L(s,χ₄) mod5 uses real character modulo 5 (χ(n)=±1 for n≡±1,±2 mod 5) and L(s,χ_4) mod4 uses character modulo 4 (χ(n)=±1 for n≡1,3 mod 4), consistent with dataset description requirements. - **L_DH implementation:** Confirmed L_DH coefficients are zero for non-squarefree numbers (a_n = μ²(n)·λ(n)), consistent with corrected implementations post-r17/r52/r58. - **Coefficient generation:** Generated new coefficient sequences for L(s,χ₄) mod5 and L(s,χ_4) mod4 up to N=10⁵; used existing coefficients from .npy files for ζ, f_rand, f_alt (=L_DH), and f_fully_rand.
