## Overview ## FINAL ANSWER The research hypothesis is **REJECTED**. The GEV shape parameter ξ of the R_comp metric distribution does **NOT** reliably distinguish "standard" L-functions from "anomalous" ones in the hypothesized manner. ### Key Findings **1. Statistical Distinguishability:** The four function classes show statistically significant differences in their GEV shape parameters (one-way ANOVA: F = 4415.5, p < 1e-200). When grouped by type, anomalous functions (mean ξ = -0.210) differ significantly from standard functions (mean ξ = -0.259) with t = 19.05, p < 1e-76, and Cohen's d = 0.60. **2. Contrary to Hypothesis:** The hypothesis predicted that anomalous functions would exhibit **heavier tails** (higher ξ values). The opposite was found: anomalous functions have ξ values **closer to zero** (less negative), indicating **lighter tails** and **greater variability** in extreme values, while standard functions have more negative ξ values, indicating more **bounded and tightly concentrated** extreme distributions. **3. Specific Results:**
- **ζ(s) (Standard):** ξ = -0.296, 95% CI: [-0.361, -0.236]
- **L(s, μ) (Anomalous):** ξ = -0.112, 95% CI: [-0.203, -0.045] — **Clearly separated from ζ**
- **L(s, λ) (Anomalous):** ξ = -0.297, 95% CI: [-0.390, -0.219] — **Overlaps with ζ**
- **L(s, χ₄) (Standard):** ξ = -0.220, 95% CI: [-0.310, -0.124] All functions exhibit negative ξ (Weibull-type distributions), not the Fréchet-type (ξ > 0) expected for heavy-tailed behavior. **4. Inconsistent Classification:** Only L(s, μ) (Möbius function, with poles at ζ zeros) shows a ξ value clearly distinct from ζ. L(s, λ) (Liouville function, the validated historical L_DH from r77) has confidence intervals that overlap with ζ, making it indistinguishable from standard functions using this metric alone. **5. Conclusion on Reliability:** The ξ parameter is **NOT a reliable universal classifier** for analytic anomalies. While it can detect L(s, μ), it fails to consistently identify anomalous functions, and the direction of the effect contradicts the theoretical expectation. The tail-heaviness of the R_comp distribution does not provide a robust signal of off-line zeros or poles. --- ## DISCRETIONARY DECISIONS - **Function selection:** Used four archetypes based on dataset description: ζ(s), L(s, μ), L(s, λ), and L(s, χ₄). The choice of L(s, λ) as "validated historical L_DH" follows the dataset description stating r77 used λ(n) coefficients with anomalous behavior. - **R_comp metric definition:** Defined R_comp(t) as the ratio of local maximum to local RMS in a sliding window (window size = 100 points, spanning Δt × 100 = 50 in t-space). This captures local resonance strength relative to typical fluctuation, consistent with resonance complexity measurement. - **Truncation length:** Used N = 10^5 as specified in the research objective and confirmed as practical in the dataset description. - **Time range and sampling:** Used t ∈ [1000, 20000] with Δt = 0.5, providing denser sampling than the standard 2π/log(N) ≈ 0.546 for better R_comp calculation. - **Block maxima extraction:** Used exactly 200 blocks as specified in the research objective, with block size = 37901/200 ≈ 189 points per block. - **GEV fitting method:** Used Maximum Likelihood Estimation (scipy.stats.genextreme.fit) with bootstrap resampling (1000 samples) for confidence interval estimation, which is more robust than asymptotic standard errors for finite samples. - **Significance threshold:** Set α = 0.05 for all statistical tests. - **Coefficient generation:** Generated Möbius μ(n) and Liouville λ(n) using standard sieve algorithms. For the real character mod 5, used the residue pattern χ₄(n mod 5) = {1, -1, -1, 1, 0}. - **Bootstrap seed:** Used seed = 42 for reproducibility of bootstrap confidence intervals. - **Statistical tests:** Applied Welch's t-test (unequal variances) rather than Student's t-test, appropriate when group variances differ. - **Effect size metric:** Used Cohen's d for standardized effect size measurement, calculated as the difference in means divided by pooled standard deviation.
