## Overview The hypothesis that the "Anomalous-Resonant" class would have a GEV shape parameter ξ statistically distinct from both "Standard" and "Control" classes is **not supported** by the data. One-way ANOVA revealed no significant differences between the three classes (F=0.479, p=0.651), and all post-hoc pairwise comparisons were non-significant (all p > 0.36). While effect sizes between Standard and the other two classes were large (Cohen's d ≈ -0.84 to -0.88), high within-class variability prevented statistical significance. The Anomalous-Resonant class showed substantial heterogeneity: L(s,λ(n)) exhibited positive ξ = 0.062 [-0.033, 0.154], L(s,μ(n)) had near-zero ξ = -0.060 [-0.159, 0.029], and L_DH showed negative ξ = -0.172 [-0.330, -0.097]. Confidence intervals overlapped for 17 of 21 function pairs. These findings confirm that GEV shape parameters of R_comp metrics do not reliably separate standard L-functions from those with analytic anomalies, consistent with prior research (r84). **DISCRETIONARY DECISIONS:** • **R_comp metric definition**: Defined as R_comp(t) = |D_F(t)| / <|D_F|>_local using a sliding window (window_size=100 points). Alternative definitions (e.g., ratio to global mean, different window sizes) could be explored. • **Sliding window size**: Set at 100 points for local statistics computation in R_comp. This represents approximately 55 units in t-space given dt ≈ 0.546. • **Block size selection**: Used 200 blocks for extracting maxima, yielding approximately 174 points per block. Different block sizes would affect the GEV fitting but not fundamentally alter class comparisons. • **Bootstrap iterations**: Used 1000 bootstrap samples for confidence interval estimation. Standard practice; more samples (e.g., 10,000) would provide marginally tighter estimates. • **Significance threshold**: Applied α = 0.05 for all statistical tests (ANOVA and post-hoc comparisons). • **GEV fitting method**: Used maximum likelihood estimation via scipy.stats.genextreme. Alternative methods (L-moments, method of moments) exist but MLE is standard for GEV. • **Post-hoc test choice**: Used Welch's t-test (does not assume equal variances) rather than standard t-test or Tukey's HSD. Welch's test is more robust to unequal variances and appropriate for unequal sample sizes. • **Time range selection**: Analyzed t ∈ [1000, 20000] as specified in objectives. Lower bound excludes potential edge effects; upper bound determined by computational feasibility. • **Sampling resolution**: Used dt = 2π/log(N) as standard resolution from dataset description, yielding 34,815 time points. • **Random seeds**: Used seed=42 for both f_rand and f_fully_rand as specified in objectives. • **Validated L_DH implementation**: Used the Davenport-Heilbronn function definition from the research program PDF (complex character mod 5 with κ parameter), which produces real periodic coefficients with period 5. • **Coefficient computation method**: Implemented multiplicative functions using standard sieve algorithms. For Liouville λ(n) and Möbius μ(n), used prime factorization approaches. • **Numerical stability**: Applied Kahan compensated summation for computing Dirichlet series D_F(t;N) to minimize floating-point accumulation errors. • **Truncation length**: Set N=10^5 as specified in objectives, balancing computational feasibility with numerical precision.
