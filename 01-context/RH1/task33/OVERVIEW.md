I generated partial sum values D(t; N) = Σ_{n≤N} a_n / n^(1/2 + it) for both the Riemann zeta function (a_n = 1) and a random multiplicative function f_rand (seed=42, with a_p = ±1 uniform i.i.d. at primes, extended multiplicatively) using N=10^5 over t ∈ [1000, 10000]. The random multiplicative coefficients were generated using a sieve to identify primes, randomly assigning ±1 values at each prime, then extending multiplicatively via prime factorization. All partial sums were computed using Kahan compensated summation to ensure numerical precision (relative error < 10^-12), evaluating 9,000 equally-spaced t-values. The t-range was divided into 100 blocks of 90 points each, and the maximum |D(t)| was extracted from each block, yielding 100 block maxima per function. Generalized Extreme Value (GEV) distributions were fitted to these maxima using maximum likelihood estimation (scipy.stats.genextreme). The GEV distribution has CDF F(x) = exp(-(1 + ξ(x-μ)/σ)^(-1/ξ)) with location μ, scale σ, and shape ξ parameters. Bootstrap confidence intervals (1,000 resamples) were computed for all parameters. Statistical significance was assessed via: (1) permutation test (10,000 permutations) for H₁: ξ_rand > ξ_zeta; (2) parametric bootstrap test (1,000 samples) under the null hypothesis of equal shape parameters; (3) checking if 95% CI for ξ_zeta contains 0 (GMC prediction). Goodness of fit was evaluated using Kolmogorov-Smirnov tests and moment comparisons. Return levels were computed for various return periods to characterize tail behavior.
