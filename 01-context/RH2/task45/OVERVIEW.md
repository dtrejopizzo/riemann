## Overview ## Main Finding **The hypothesis is REJECTED.** The specific, regular distribution of integers according to their ω(n) value (as described by Sathé-Selberg theorems) does **not** appear to be essential for creating stable inter-S_k phase relationships observed for ζ(s) at N=10^6 and t≈10,000. ## Quantitative Results At the primary resonance peak (t* = 10025.65, |D_ζ(t*)| = 16.76) with N=10^6: **Phase Difference Tests (1000 permutations):**
- Δθ₁ (S₁→S₂): baseline = -9.80°, null mean = 0.98° ± 43.10°, **p = 0.773**
- Δθ₂ (S₂→S₃): baseline = -7.60°, null mean = -0.94° ± 20.58°, **p = 0.691**
- Δθ₃ (S₃→S₄): baseline = 32.03°, null mean = 0.83° ± 24.54°, **p = 0.185**
- Δθ₄ (S₄→S₅): baseline = -19.94°, null mean = -2.37° ± 55.32°, **p = 0.665** None of the four phase differences showed statistical significance (all p > 0.05). The baseline values from true ζ(s) fell well within the distributions obtained by randomly shuffling ω(n) labels. **Global Coherence Metrics (500 permutations):**
- Alignment Ratio R = |Σ S_k| / Σ|S_k|: baseline = 0.9903, null = 0.9572 ± 0.0264, **p = 0.068, Z = 1.25**
- Angular Spread: baseline = 10.75°, null = 25.18° ± 13.53°, **p = 0.078, Z = -1.07** Both metrics showed weak trends (R slightly higher, angular spread slightly lower for true ζ(s)) suggesting marginally more coherent phase relationships, but neither reached statistical significance at α = 0.05. **Cross-Peak Consistency (5 peaks tested):**
- 0/5 peaks showed significant angular spread differences (p < 0.05)
- Mean Z-score across peaks: -0.13
- Z-scores ranged from -0.95 to +1.91, indicating high variability and no consistent pattern ## Interpretation At this truncation depth (N=10^6) and height range (t≈10,000–11,000), the phase relationships between S_k vectors in ζ(s) are **not significantly different** from those that would arise if ω(n) labels were randomly reassigned to integers. The oscillatory term n^(-it) appears to dominate the phase structure, with the specific number-theoretic regularity of the Sathé-Selberg distribution playing a negligible role in determining inter-S_k phase differences. The weak (non-significant) trends in alignment ratio and angular spread suggest that arithmetic regularity may have a subtle influence on overall phase coherence, but this effect is not strong enough to be detected reliably with this permutation test at the tested scale. The high variability across different resonance peaks further indicates that any such effect is not universal or consistent. ## Implications These findings suggest that: 1. **The stable phase relationships observed in r26 are NOT primarily caused by the regular ω(n) distribution.** Other mechanisms must be responsible for the inter-S_k phase structure in ζ(s). 2. **The Poisson-like profile of S_k magnitudes** (described by Selberg-Delange theorems) may be important for magnitude scaling and extremal statistics, but it does not create a distinctive phase signature at this scale. 3. **The effect of arithmetic regularity may be scale-dependent.** It could become detectable at larger N (e.g., N > 10^7) or manifest in different observables such as phase evolution with varying N or t. 4. **Phase stability at fixed (N,t) is governed primarily by the complex exponential n^(-it)** rather than by number-theoretic class structure. --- ## Discretionary Analytical Decisions - **Statistical significance threshold**: Set α = 0.05 for all hypothesis tests
- **Number of permutations**: Used 1000 permutations for phase difference analysis (computationally feasible, provides p-value resolution of ~0.001)
- **Coherence sample size**: Used 500 permutations for alignment ratio and angular spread analysis (balance between statistical power and computational cost)
- **Peak selection**: Identified resonance peaks with minimum |D| = 10.0 and minimum separation of 20 grid points in t-space
- **Phase wrapping convention**: Wrapped phase differences to [-π, π] using arctan2 for consistent comparison
- **Two-tailed vs one-tailed tests**: Used two-tailed tests for phase differences (no directional hypothesis), one-tailed test for angular spread (specific hypothesis of smaller spread indicating stability)
- **Alignment ratio metric**: Computed R = |Σ S_k| / Σ|S_k| following the established definition from r26
- **Angular spread metric**: Used circular standard deviation σ_circ = sqrt(-2 ln(R_mean)) to quantify phase dispersion
- **Random seed**: Set seed=42 for main permutation analysis, seed=43 and seed=44 for supplementary analyses (reproducibility while ensuring independence)
- **Kahan summation**: Used compensated summation for all Dirichlet series evaluations following validated protocol from r1
