## Overview ## Analysis Summary The parametric family of multiplicative functions with randomly assigned prime coefficients (a_p = +1 with probability α, -1 with probability 1-α) exhibits GEV shape parameters ξ that are largely insensitive to changes in α for α ∈ [0, 0.8], with ξ ranging from 0.226 to 0.271. However, the correlation between mean R_comp and fitted ξ across all seven α values is weak and statistically insignificant (Spearman ρ = -0.357, p = 0.432), confirming that the R_comp → ξ link is NOT universal. The most striking finding is that the ζ-like function (α = 1.0, all a_p = +1) is a structural outlier with ξ = 0.048 [95% CI: -0.053, 0.159], approximately 5 times smaller than the random ±1 functions, despite having a moderate R_comp value of 0.929. This demonstrates that extreme value behavior depends fundamentally on function structure rather than resonance suppression metrics alone. ### Complete Results Table | α | ξ | 95% CI ξ | Mean R_comp | Std R_comp |

This indicates that extreme value distributions depend on the underlying coefficient structure. 4. **Contrast with Previous Work**: Previous analyses (r17) demonstrated a strong R_comp → ξ link for surgical perturbations of ζ(s) that maintain its structural properties. This analysis shows that the link breaks down for functions with fundamentally different coefficient structures (random ±1), establishing that the relationship is specific to functions "close" to ζ(s) rather than universal. ### Methodological Notes All analyses used N = 10⁵ with mandatory Kahan compensated summation for numerical stability. GEV parameters were fitted to 200 block maxima extracted from |D_F(t)| computed over t ∈ [1000, 20000] (3000 sampled points), with 95% confidence intervals computed via 1000 bootstrap iterations. R_comp values were averaged over 50 random t values in [1000, 3000]. The scipy.stats.genextreme convention (c = -ξ) was properly handled throughout. --- ## Discretionary Decisions - **t-sampling strategy**: Used coarse sampling (3000 points) instead of Δt = 2π/log(N) spacing (~35,000 points) to meet computational constraints. This gives ~15 points per block, sufficient for reliable maxima extraction. - **Random seed selection**: Used seed=42 for reproducibility, with different seeds (42+i) for each α value to ensure independent random realizations of prime coefficients. - **R_comp t-range**: Computed R_comp over [1000, 3000] rather than matching the full GEV range [1000, 20000] to reduce computational burden while maintaining representative sampling. - **Number of R_comp samples**: Used 50 random t values for R_comp averaging, balancing computational cost with statistical reliability. - **GEV bootstrap iterations**: Set at 1000 iterations following previous protocol (r17) for stable confidence interval estimation. - **Block count**: Used 200 blocks for GEV analysis as specified in the research objective, following standard extreme value theory practice requiring n ≥ 100. - **Significance threshold**: Applied standard α = 0.05 for statistical significance testing of correlations. - **Prime coefficient assignment**: Generated multiplicative functions by first assigning ±1 to primes according to probability α, then extending multiplicatively via factorization (a_{p^k} = a_p^k, a_{nm} = a_n·a_m for coprime n,m).
