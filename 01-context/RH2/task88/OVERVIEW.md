## Overview ## Complete Analysis of Gumbel μ Prediction from S_k Covariance Features ### Main Results **Model Performance on Full Dataset (14 functions):**
- LOOCV R² = **-0.1598** (failed - model performs worse than predicting the mean)
- LOOCV RMSE = 0.4915
- Optimal Ridge alpha = 1000.0
- **Critical issue**: F_k1, F_k2, F_k3 have all-zero covariance features (C_55=0, C_66=0, all off-diagonal elements=0) because they isolate specific ω(n) classes. These functions have significantly different μ values (1.30-1.61) that cannot be predicted from the zero features, making the full 14-function dataset unsuitable for regression. **Model Performance on Subset without F_k functions (11 functions: ζ(s) + 10 random multiplicative):**
- LOOCV R² = **0.5099** (explains 51% of variance)
- LOOCV RMSE = 0.0488
- LOOCV MAE = 0.0327
- Optimal Ridge alpha = 0.01
- 30% improvement over baseline (predicting mean μ̄=2.54) **Feature Importance (Subset Model - ranked by |coefficient|):**
1. sum_negative_real_offdiag: **+0.0858** (most predictive)
2. mean_offdiag_real: +0.0088
3. C_55: +0.0021
4. std_offdiag_real: -0.0015
5. C_66: -0.000005 (negligible) **Gumbel Fitting Quality:**
All 14 functions fit well to Gumbel distribution (Kolmogorov-Smirnov test p > 0.05 for 13/14 functions). μ ranges from 1.30 (F_k1) to 2.59 (random_4). ### Hypothesis Test Result **Hypothesis:** Features measuring anti-correlation (like sum_negative_real_offdiag) will be **negatively** correlated with μ, as stronger anti-correlation should suppress the typical size of maxima. **Result:** **HYPOTHESIS REJECTED** The most important feature, sum_negative_real_offdiag, has a **strong positive coefficient** (+0.0858, largest magnitude). This means:
- MORE negative sum_negative_real_offdiag → MORE anti-correlation in S_k covariance
- But this predicts **HIGHER** μ (larger typical maxima)
- This is the **opposite** of what was hypothesized **Critical caveat:** The apparent positive correlation (Pearson r=0.925, p=4.5×10⁻⁵) in the subset is driven by **ζ(s) as an extreme outlier**:
- ζ(s): sum_neg_offdiag = -2.69 (26 std below mean), μ = 2.32
- Random functions: sum_neg_offdiag ≈ -0.12 (mean), μ ≈ 2.54 (mean)
- Among the 10 random functions alone: **NO significant correlation** between sum_negative_real_offdiag and μ (r=0.15, p=0.56) The R²=0.51 model is essentially fitting a line from the single outlier (ζ) through the cloud of random functions, not discovering a general physical law. ### Gumbel Parameters by Function Type | Function Type | μ Range | Mean μ | Key Characteristic |

| F_k functions (k=1,2,3) | [1.30, 1.61] | 1.43 | Zero covariance features |

- Only C_55 shows significant correlation with μ: Spearman r=0.64, p=0.013
- sum_negative_real_offdiag: r=-0.31, p=0.28 (not significant) **Subset without F_k (11 functions):**
- No individual feature shows significant correlation with μ (all p > 0.2)
- The model's predictive power (R²=0.51) comes from the **combination** of features, not individual relationships ### Interpretation 1. **The Gumbel location parameter μ CAN be predicted from S_k covariance features (R²=0.51 for ζ+random functions), providing moderate support for the log-correlated field analogy proposed in r41.** 2. **However, this is NOT evidence for a universal covariance-μ relationship.** The model's performance is driven by a single outlier (ζ) with extreme S_k anti-correlations and intermediate μ, versus random functions with weak correlations and high μ. 3. **Among random functions alone, there is NO predictive relationship** between covariance features and μ, suggesting that μ is relatively insensitive to small variations in S_k structure. 4. **The direction of the relationship contradicts the hypothesis:** stronger anti-correlation is associated with HIGHER μ (in the outlier-driven model), not lower μ. 5. **The F_k functions demonstrate that μ can vary substantially (1.30-1.61) even when all covariance features are zero,** indicating that factors beyond the measured covariance structure influence μ. ### Comparison to Previous Work (r40) The previous analysis (r40) successfully predicted the GEV shape parameter ξ from the same covariance features. The current analysis for Gumbel μ shows:
- **Much weaker predictive power**: R²=0.51 for μ vs higher R² for ξ (from r40)
- **Opposite relationship direction**: Anti-correlation features that suppress ξ (moving toward Gumbel) appear to increase μ
- **Greater sensitivity to outliers**: The μ model is dominated by a single outlier, while the ξ model presumably showed more general patterns This suggests that **covariance structure more strongly constrains the tail shape (ξ) than the location (μ)** of extreme value distributions. --- ## Discretionary Analytical Decisions - **Ridge regression chosen over OLS**: Given the small sample size (n=11-14) and potential multicollinearity among covariance features, Ridge regression was selected to prevent overfitting. Alternative: Lasso or Elastic Net. - **Ridge alpha selection via GridSearchCV**: Tested alphas from 0.001 to 1000 using LOOCV scoring. Optimal alpha=1000 for full dataset (heavy regularization due to poor fit), alpha=0.01 for subset (light regularization for better predictive fit). - **Leave-One-Out Cross-Validation (LOOCV) used instead of k-fold**: With only 11-14 samples, LOOCV provides maximum training data per fold and is standard for small sample regression. - **Inclusion of F_k functions in dataset**: As specified in the research objective, included F_k1, F_k2, F_k3 from r40 despite their pathological zero features. Reported results both with and without these functions. - **Block size for Gumbel fitting**: Used 100 blocks (same as r40) with 90 t-values per block (9000 total t-values). This provides stable Gumbel parameter estimates. - **t-value range**: Used t ∈ [1000, 10000] as specified in dataset description and r40, avoiding lower t where finite-N effects dominate. - **Significance threshold**: Used α=0.05 for all statistical tests (standard convention). - **Kolmogorov-Smirnov test for Gumbel goodness-of-fit**: Standard test for assessing whether block maxima follow the fitted Gumbel distribution. - **Spearman correlation for robustness**: Used Spearman rank correlation in addition to Pearson to assess monotonic relationships without assuming linearity, appropriate given small sample size and potential outliers. - **Random function seeds**: Used seeds 101-110 for the 10 random multiplicative functions to ensure reproducibility while maintaining independence from other analyses. - **N=10⁵ truncation depth**: As specified in r40, used N=100,000 for computational feasibility while maintaining sufficient arithmetic structure in S_k sums.
