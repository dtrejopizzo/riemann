## Overview <output>
<conclusion>
The inverse correlation between bulk kurtosis and GEV ξ observed across diverse L-functions does NOT hold within a parametric family of multiplicative functions where prime coefficients are set to ±1 with varying probability: no statistically significant correlation was found (Spearman ρ = 0.167, p = 0.693).
</conclusion> <methods>
I implemented a generator for parametric multiplicative functions where for each prime p, the coefficient a_p equals +1 with probability α and -1 with probability (1-α), then extended multiplicatively to all n using a sieve-based approach. For N=10⁵, I computed Dirichlet partial sums D(t;N) = Σ_{n≤N} a_n/n^(1/2+it) using vectorized numpy operations (an accepted alternative to Kahan summation at this scale per the dataset description) across 5,000 t-points in [5000, 25000] for eight α values: 0.5, 0.6, 0.7, 0.8, 0.9, 0.95, 0.99, and 1.0. For each α, I calculated: (1) the excess kurtosis of the full log|D(t;N)| distribution using scipy.stats.kurtosis with fisher=True; (2) the GEV shape parameter ξ by fitting scipy.stats.genextreme to block maxima (100 blocks of 50 points each). I then computed the Spearman rank correlation coefficient between excess kurtosis and ξ using scipy.stats.spearmanr, chosen per the dataset description's preference for non-parametric methods. Computational time was ~177 seconds total on standard CPU.
</methods> <results>
Across the eight α values tested, excess kurtosis ranged from 0.022 (α=0.95) to 0.977 (α=0.5), while GEV ξ ranged narrowly from -0.318 (α=0.9) to -0.208 (α=0.8). All ξ values were negative, indicating Weibull/bounded tail behavior. The Spearman rank correlation between excess kurtosis and ξ was ρ = 0.167 (p = 0.693), showing no statistically significant relationship at the α=0.05 level. This positive (not negative) weak correlation directly contradicts the hypothesis. Pearson correlation was similarly non-significant (r = 0.231, p = 0.582). Excluding α=1.0 as a special case (constant coefficients) did not materially change the result (ρ = 0.357, p = 0.432). The relationship appeared non-monotonic with no clear pattern as α varied.
