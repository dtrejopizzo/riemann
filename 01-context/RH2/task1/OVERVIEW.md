## Overview # ANSWER The hypothesis that ζ(s) would show greater sensitivity to block size in GEV analysis—due to strong correlations causing slow convergence to the asymptotic extreme value distribution—is **NOT SUPPORTED** by this analysis. ## Quantitative Evidence **1. Block-Size Sensitivity (Linear Regression of ξ vs log(Block Length)):** - **ζ(s)**: slope = -0.045 ± 0.022, R² = 0.58, p = 0.136
- **f_rand**: slope = -0.065 ± 0.021, R² = 0.76, p = 0.056
- **Slope ratio**: |ζ| / |f_rand| = 0.70
- **Statistical test for difference**: Z = 0.626, p = 0.531 (not significant) Neither function shows statistically significant block-size dependence at α = 0.05. Critically, f_rand (the random multiplicative function expected to have weaker correlations) shows *slightly stronger* block-size sensitivity than ζ(s), directly contradicting the hypothesis. **2. - Block-size sensitivity appears to be a general finite-sample phenomenon, not uniquely diagnostic of ζ(s) correlations ## Conclusion The research hypothesis is rejected. The anticipated signature—greater block-size sensitivity in ζ(s) due to strong correlations—is not observed. Instead, both functions show similar (non-significant) sensitivity, with f_rand showing marginally stronger dependence. This suggests that: 1. The observed block-size effects reflect general pre-asymptotic behavior common to both functions
2. Block-size sensitivity alone cannot distinguish correlation-driven convergence issues from other finite-sample effects
3. The robust domain separation (Weibull vs. Fréchet) provides a clearer discriminant between function classes than block-size sensitivity The results are consistent with both functions operating in pre-asymptotic regimes at these block sizes (10-100 points), but the mechanism driving block-size dependence is not uniquely attributable to ζ(s) correlations. --- # DISCRETIONARY ANALYTICAL DECISIONS - **Block size selection**: Used B ∈ {400, 200, 100, 50, 40} blocks (lengths 10, 20, 40, 80, 100 points). This represents 1:10 subsampling of the original design. - **Confidence interval method**: Used 1000-sample bootstrap resampling for 95% CI estimation rather than asymptotic Hessian-based intervals. Rationale: Bootstrap is more robust for small sample sizes (B=40, 50) and makes no distributional assumptions about the GEV parameter estimates. - **Random multiplicative function seed**: Used seed=42 for f_rand as specified in research objective. - **Regression model**: Linear regression of ξ vs log(Block Length) rather than ξ vs Block Length.
