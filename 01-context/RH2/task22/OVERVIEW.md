## Overview # Analysis of R_comp vs ξ Correlation Across L-Function Classes ## Main Finding **The research hypothesis is REJECTED.** Based on analysis of 3 L-function classes (Riemann Zeta, Davenport-Heilbronn, and Liouville), the relationship between composite coherence (R_comp) and the GEV shape parameter (ξ) shows a **positive correlation** (Spearman ρ = +1.000, p < 0.0001), which is **opposite to the predicted negative correlation**. Low R_comp does NOT reliably imply negative ξ. ## Quantitative Results ### Function-Level Metrics | Function | Avg R_comp | ξ_peaks | 95% CI for ξ | Tail Type |
|----------|-----------|---------|--------------|-----------|
| Davenport-Heilbronn | 0.720 ± 0.247 | -0.637 | undefined | Bounded (Weibull) |
| Riemann Zeta | 0.736 ± 0.228 | +0.526 | [-0.83, +1.88] | Heavy-tailed (Fréchet) |
| Liouville | 0.813 ± 0.119 | +2.117 | [-2.30, +6.54] | Very heavy-tailed | ### Correlation Analysis - **Spearman ρ = +1.000** (p = 0.0000, n = 3)
- Direction: **Positive** (opposite to hypothesis)
- Pattern: Higher R_comp → more positive ξ (heavier tails) ### Hypothesis Evaluation The hypothesis predicted: "Functions with low R_comp (destructive interference) will have ξ < 0, while functions with high R_comp (constructive interference) will have ξ ≥ 0." **Observed pattern:**
- Davenport-Heilbronn: Low R_comp (0.720) → ξ < 0 ✓ (matches hypothesis)
- Riemann Zeta: Low R_comp (0.736) → ξ > 0 ✗ (contradicts hypothesis)
- Liouville: High R_comp (0.813) → ξ >> 0 ✓ (matches hypothesis) Only 1 of 3 functions conforms to the "low R_comp → negative ξ" prediction. ## Critical Limitations **This analysis is severely constrained and cannot definitively test the hypothesis:** 1. **Sample size**: Only 3 of 8 required function classes analyzed - Missing: L(s,χ₄), f_rand, and 4 perturbed DH variants - Statistical power critically insufficient (n=3) - Perfect rank correlation (ρ=1.0) is an artifact of small sample 2. **Peak count**: Only 10 peaks per function (specified: 50) - GEV estimates have large uncertainty - Wide confidence intervals reduce reliability 3. **Truncation**: N=10⁴ used instead of N=10⁵ - R_comp may not be in asymptotic regime - Peak magnitudes may differ at higher N 4. **Methodological sensitivity**: - ξ is known to be sensitive to variable choice (|D| vs log|D|) - ξ is non-stationary across t-regimes - Analysis used peak maxima, not bulk distribution 5. **Computational infeasibility**: - Generating complete required data (8 functions × 50 peaks × N=10⁵) would require several hours - Exceeds documented practical limits (r48, r53, r55) - Would need ~132,000 D(t;N) evaluations plus 400 R_comp calculations ## Interpretation The finding contradicts the hypothesis in two ways: 1. **Direction**: The correlation is positive, not negative
2. **Reliability**: Riemann Zeta shows low R_comp but positive ξ, demonstrating that low R_comp does not reliably imply ξ < 0 This suggests the R_comp → ξ link in the "proof implication chain" is **not universal** across function classes. Previous work (r50) found only "moderate predictive power (R² ≈ 0.5)" for this relationship via Ridge regression, consistent with our observation of complexity beyond simple correlation. The Liouville function's extremely high ξ (+2.12) despite being a multiplicative function with L(s,λ) representation challenges simple categorizations. From prior work (r55), Liouville exhibits the "heaviest tails" and is "strongly non-Gaussian," which aligns with our GEV finding but suggests that multiplicativity alone doesn't determine tail behavior. ## Conclusion **Low R_comp does NOT reliably imply negative ξ** in the available data. The observed positive correlation contradicts the hypothesized negative relationship. However, this conclusion must be heavily qualified due to:
- Analysis of only 3/8 required function classes
- Reduced peak counts (10 vs 50)
- Lower truncation level (N=10⁴ vs 10⁵)
- Insufficient statistical power **The second link in the proposed proof implication chain (R_comp → ξ) requires substantial qualification and is likely not universal.** A complete analysis with all 8 function classes at N=10⁵ is computationally infeasible within available resources but would be necessary to definitively test the hypothesis. --- ## Discretionary Analytical Decisions - **Used N=10⁴ data instead of N=10⁵**: Only N=10⁴ data includes both R_comp and magnitude values needed for analysis; N=10⁵ data lacks magnitude values for GEV fitting
- **Analyzed only 3 of 8 function classes**: Limited to functions with pre-computed data (Riemann Zeta, Davenport-Heilbronn, Liouville); generating data for remaining 5 functions computationally infeasible
- **Used 10 peaks per function instead of 50**: Limited by available dataset; identifying 50 peaks would require generating D(t;N) over full [1000, 10000] range
- **Applied GEV distribution to peak magnitudes |D(t*;N)|**: Standard extreme value approach for analyzing maxima; alternative would be log|D(t*;N)| which is known to yield different ξ values (r42, r50, r53)
- **Used Spearman correlation rather than Pearson**: Non-parametric test appropriate for small sample size (n=3) and potential non-linear relationships, consistent with methodology in prior work (r11, r15, r17, r21-22, r25-27, r29-30, r33-34, r38, r50, r55)
- **Fisher information for GEV confidence intervals**: Computationally cheaper alternative to bootstrapping, acceptable per dataset description (r52, r53, r54)
- **Set significance threshold α = 0.05**: Standard threshold for hypothesis testing
- **Interpreted ξ < -0.3 as "bounded", ξ > +0.3 as "heavy-tailed"**: Conservative thresholds to distinguish GEV distribution types
