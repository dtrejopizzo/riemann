## Overview ## ANSWER The research hypothesis is **REJECTED**. The predicted negative correlation between mean R_comp and tail-lightness does NOT hold for the parametric multiplicative family examined. ### Quantitative Evidence **Data generation:**
- Analyzed 11 α values from 0.0 to 1.0 (step 0.1)
- For each α: generated multiplicative coefficients with a_p = +1 w.p. α, -1 w.p. (1-α)
- Computed 300 |D_F(t; N)| values at random t ∈ [1000, 5000] with N = 10⁴
- Calculated mean R_comp and fitted tail models for each α **Tail model preferences:**
- Power law (heavy tail) preferred: 9/11 cases (81.8%)
- Stretched exponential (light tail) preferred: 2/11 cases (18.2%, at α = 0.7 and 0.9) **R_comp comparison:**
- Mean R_comp for power law cases: 0.011803 ± 0.000762
- Mean R_comp for stretched exponential cases: 0.012477 ± 0.001002
- **Direction:** Stretched exponential cases have HIGHER R_comp (opposite of hypothesis)
- **Statistical test:** t = -0.9610, p = 0.3616 (not significant) **Correlation analysis:**
- Spearman correlation (R_comp vs stretched_exp preference): ρ = 0.0745, p = 0.8276
- **Direction:** Positive (opposite of hypothesized negative correlation)
- **Strength:** Negligible (|ρ| = 0.07) **Model selection strength:**
- Power law cases show strong preference: ΔAIC ranges from 12.5 to 79.7
- Stretched exponential cases show weak preference: ΔAIC = 2.07 and 0.07
- Five cases have ΔAIC > 50, all favoring power law ### Key Findings 1. **No support for hypothesized correlation:** The data show the opposite trend - stretched exponential cases have slightly higher (not lower) R_comp, but this difference is not statistically significant (p = 0.36). 2. **Dominant heavy-tail behavior:** The parametric family predominantly exhibits power law (heavy) tails across most α values, contradicting the expectation that varying multiplicativity structure would produce a spectrum of tail behaviors. 3. **Weak preference for light tails:** Only 2 of 11 cases prefer stretched exponential, and both have marginal ΔAIC values (< 3), suggesting ambiguous tail classification rather than clear stretched exponential behavior. 4. **R_comp shows minimal variation:** Mean R_comp ranges only from 0.010 to 0.013 across all α values (30% range), with large overlapping error bars, indicating R_comp is relatively insensitive to the parameter α in this family. 5. **No clear α-dependent trend:** R_comp shows weak negative correlation with α (r = -0.26), but tail preference shows no systematic pattern with α. ### Conclusion The "bridge hypothesis" from r9 - that low R_comp correlates with light (stretched exponential) tails - is **not supported** by this parametric family analysis. The multiplicative functions examined predominantly exhibit heavy (power law) tails regardless of R_comp values, which remain consistently low (~0.012) across all α. The two cases preferring stretched exponentials have higher, not lower, R_comp values, and this difference is not statistically significant. This suggests that R_comp alone does not reliably predict tail behavior in this multiplicative family, and additional structural features beyond composite coherence may govern tail properties in arithmetic L-functions. --- ## DISCRETIONARY DECISIONS - **N = 10⁴ instead of 10⁵:** Reduced from original specification due to computational constraints. This is an acceptable pilot study approach per the dataset description (r6 strategy), though results at N=10⁵ may differ. - **300 samples instead of 2000:** Reduced sample size from 2000 to 300 per α value to achieve feasible runtime. While this reduces statistical power, 300 samples per condition is still adequate for mean estimation and model fitting. - **Top 10% for tail fitting:** Used the 90th percentile threshold to define the upper tail for power law and stretched exponential fitting, a standard choice in extreme value analysis. - **Survival function regression approach:** Fitted power law using log-log linear regression on empirical survival function, and stretched exponential using nonlinear least squares. Alternative approaches (maximum likelihood, Hill estimator) could yield different parameter estimates. - **AIC for model selection:** Used Akaike Information Criterion rather than BIC for model comparison. Both are valid; BIC penalizes complexity more heavily and could change marginal cases. - **Standard (not Kahan) summation for D_F:** Used standard numpy summation for the 300×11 = 3300 partial sum evaluations rather than Kahan compensated summation to improve computational speed. Validation showed this introduces acceptable error for N=10⁴ and the t-range used. - **Nelder-Mead optimization for stretched exponential:** Used Nelder-Mead simplex method for nonlinear fitting of stretched exponential parameters. Alternative optimizers (L-BFGS-B, differential evolution) could be more robust but are slower. - **t-range [1000, 5000]:** Used the specified range from the research objective. This represents the accessible computational regime per the dataset description. - **Seed-based reproducibility:** Used α-dependent seeds (seed = int(α*100) + 1000) to generate different coefficient realizations for each α while maintaining reproducibility. - **Spearman vs Pearson correlation:** Reported Spearman rank correlation as the primary correlation measure due to small sample size (n=11) and potential non-normality, though Pearson correlation gives similar results. - **Alpha = 0.05 significance threshold:** Used standard α = 0.05 threshold for statistical tests, though this is somewhat arbitrary for exploratory analysis.
