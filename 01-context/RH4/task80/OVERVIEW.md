## Overview <output>
<conclusion>
Peak-level instantaneous spectral features derived from m_{jk}(t)=Re(S_j(t)conj(S_k(t))) do not provide a viable standalone classifier under the strict protocol, achieving held-out AUC 0.587 (95% bootstrap CI 0.538–0.636), well below the baseline magnitude-only model (AUC 0.828).
</conclusion>
<methods>
I analyzed `peaks_features_F1_F12_normalized_full.csv` (2200 rows; 11 classes × 200 peaks) together with the local coefficient and intermediate-array files in the workspace. First, I performed data validation: checked dataset dimensions and class balance, confirmed that the CSV contains 200 peaks per function class, and identified that the existing local complex `S_k` `.npy` files did not align with the CSV peak list, so I regenerated complex `S_k(t)` values from scratch for the CSV peak locations. I read the PDF specification to recover the function definitions F1–F12 and then reconstructed class coefficients as follows: F1 with `a_n=1`; F2 as the primitive order-4 Dirichlet character mod 5; F4 as the Davenport–Heilbronn periodic coefficient pattern `(1, κ, −κ, −1, 0)` with κ=0.28408; F5± as the same pattern with κ±0.05; F6 as Liouville `λ(n)=(-1)^Ω(n)`; F7 as Möbius `μ(n)`; F9 (Ramanujan Δ) and F10 (level-11 weight-2 newform) from `cypari2`/PARI `mfcoefs`; F11 from the local normalized coefficients in `lambda_n.npy` after verifying they are Sym²(Δ), not Δ, by checking `b_p=τ(p)^2-p^11`; and F12 from local complex coefficients in `a.npy`. I computed `ω(n)` (distinct prime factors) and `Ω(n)` (prime factors with multiplicity) up to N=10^6 using Numba-accelerated sieve-based routines. Then I evaluated, for each class and each CSV peak t, the stratified complex sums
`S_k(t)=Σ_{n≤N, ω(n)=k} a_n n^(-1/2-it)`
using Numba-jitted functions over all 8 strata `k=0..7`. For modular objects I used the analytically normalized coefficients (`a_n/n^(11/2)` for F9, `a_n/sqrt(n)` for F10, `b_n/n^11` for F11). I validated the regeneration by comparing `|S_k|` against the CSV magnitudes; F1, F4, F9, and F12 matched with median relative errors about 10^-14 to 10^-15 and maximum relative errors below 5.4×10^-10. For each of the 2200 peaks, I constructed the 8×8 real symmetric instantaneous matrix `m_{jk}(t)=Re(S_j(t)conj(S_k(t)))`, computed its eigenvalues with `numpy.linalg.eigvalsh`, clipped tiny negative numerical values to zero, and derived two features: `pc1_instantaneous` (largest eigenvalue divided by total eigenvalue sum) and `H_instantaneous` (Shannon entropy of the normalized eigenvalue spectrum, in nats). I merged these into an augmented feature table. For classification, I followed the strict protocol: training on F1, F4, F9 and testing on F2, F6, F7, F10, F11, F12. The binary target was GRH-violator status (`1` for violator, `0` otherwise). I trained an RBF-kernel SVM in a `scikit-learn` pipeline with `StandardScaler` and `SVC(kernel='rbf', C=1.0, gamma='scale')`. I evaluated AUC using the held-out decision scores and estimated 95% bootstrap confidence intervals with 1000 resamples of the held-out set. For comparison, I reproduced the documented baseline using normalized `log|S_k|` features for k=1..6, excluding `S_7` because of the known mod-5 leakage at N=10^6. The analysis used Python, NumPy, pandas, Matplotlib, Numba, SciPy, scikit-learn, and cypari2/PARI. Final outputs were saved to disk, including the reconstructed complex arrays, augmented feature tables, and a two-panel summary figure.
</methods>
<results>
The regenerated complex `S_k(t)` values reproduced the CSV magnitudes with essentially exact numerical agreement: median relative errors were 8.60×10^-15 for F1, 3.74×10^-15 for F4, 8.11×10^-15 for F9, and 4.51×10^-15 for F12; the corresponding maximum relative errors were 2.92×10^-10, 8.80×10^-11, 5.36×10^-10, and 1.75×10^-10. Using only the two new instantaneous spectral features, the strict-protocol RBF SVM achieved held-out AUC = 0.587. The 95% bootstrap CI was 0.538 to 0.636 (1000 bootstrap resamples). This is substantially worse than the established baseline. The reproduced baseline model using normalized `log|S_k|` for k=1..6 yielded held-out AUC = 0.828 with 95% bootstrap CI 0.803 to 0.850, matching the previously documented benchmark (~0.828). Adding the instantaneous spectral features to the baseline feature set reduced performance slightly rather than improving it: combined-feature AUC = 0.804 with 95% bootstrap CI 0.781 to 0.828. Class-wise averages of the instantaneous features showed only partial separation. Mean `pc1_instantaneous` / `H_instantaneous` by class were: F1 0.974 / 0.108, F2 0.969 / 0.124, F4 0.956 / 0.165, F6 0.954 / 0.155, F7 0.947 / 0.168, F9 0.971 / 0.119, F10 0.967 / 0.127, F11 0.971 / 0.117, F12 0.941 / 0.204. The key failure mode is that RH-respecting F6 and F7 overlap strongly with DH-type violators F4/F5± in this feature space. At the class level, however, the signal was present. The class-averaged spectral features from `M_jk = E[Re(S_j conj(S_k))]` were clearly separated: for example, F1/F2/F9/F10/F11 had `pc1_class` about 0.953–0.959 and `H_class` about 0.212–0.222, whereas F4/F5± had `pc1_class` about 0.922–0.924 and `H_class` about 0.361–0.368, and F12 had `pc1_class` 0.903 and `H_class` 0.393. The mean instantaneous features correlated strongly with these class-level spectral features across classes: Pearson r = 0.870 for `pc1` (p = 4.91×10^-4) and r = 0.896 for entropy (p = 1.86×10^-4). A key structural finding is that each instantaneous matrix `m(t)` is numerically rank-2 (two nonzero eigenvalues, six near-zero eigenvalues), because it is the real part of a single complex outer product. This sharply limits the spectral degrees of freedom available at the single-peak level.
</results>

I therefore could not simply merge those files with the main feature table and instead regenerated all complex `S_k` values for the CSV peak times. `cypari2` was initially unavailable and had to be installed. Generating F9 and F10 modular-form coefficients to N=10^6 was computationally expensive, and PARI memory had to be increased to 4 GB. Another provenance issue was that the local `a_n.npy`/`lambda_n.npy` files were not F9 as first suspected; they were identified as F11 (Sym²Δ) by checking prime coefficients against `τ(p)^2-p^11`. Methodologically, the instantaneous matrix definition itself imposes a strong information bottleneck: because `m(t)` is rank ≤ 2, any eigenvalue-spectrum summary can only capture a very low-dimensional projection of the full complex `S_k` geometry. This likely explains why the instantaneous features correlate with class-level structure yet fail as single-peak discriminators.
</challenges>
<discussion>
The hypothesis was only partially supported. The new instantaneous spectral features do correlate with the spectral properties of the class-averaged matrices, but that correlation does not translate into useful peak-level classification under the strict held-out protocol. In practical terms, the class-level spectral effect discovered previously is real, but it is not operationally recoverable from a single peak through the specific matrix `m_{jk}(t)=Re(S_j(t)conj(S_k(t)))` summarized only by top-eigenvalue fraction and entropy. Scientifically, this suggests that the discriminative information in the successful class-level spectral construction is an aggregate phenomenon requiring averaging across peaks. Single-peak realizations are too noisy, and the chosen instantaneous matrix is too constrained, to preserve enough separation between GRH-respecting and RH-violating families. The overlap of Liouville/Möbius classes with Davenport–Heilbronn-type violators is especially important: these classes differ in RH status but share similar instantaneous spectral compactness, undermining discriminability. Therefore, the result argues against pursuing these two peak-level eigenvalue features alone as the main path to a practical classifier. A more promising direction would be to retain richer peak-level phase information from the full complex `S_k` vector, or to use local temporal neighborhoods of peaks rather than a single observation, so that the class-level spectral structure can emerge without requiring full cross-peak averaging.
</discussion>
<proposed-next-hypotheses>
1. Peak-level classifiers built from the full complex `S_k` geometry (for example, phase-alignment features or projections onto class-derived spectral directions) will outperform the rank-2 instantaneous spectrum summaries `pc1_instantaneous` and `H_instantaneous` under the strict protocol.
2. Short-window local averages of `m_{jk}(t)` over neighboring peaks, rather than single-peak matrices, will recover enough of the class-averaged spectral structure to improve held-out AUC beyond the 0.828 magnitude-only baseline.
</proposed-next-hypotheses>
