</conclusion> <methods>
Analysis was performed using Python 3 with NumPy, SciPy, matplotlib, and pickle. The workflow consisted of five steps: 1. Function Definition: Defined f_canon_rand as a completely multiplicative function with |f(p)| = 1 for all primes p and arg(f(p)) ~ Uniform[0, 2π) with seed=42. Generated 78,498 primes up to 10⁶ using the Sieve of Eratosthenes. Assigned random phases to primes using np.random.seed(42). Computed f(n) for all n ≤ 10⁶ using a smallest-prime-factor sieve approach, exploiting the multiplicative property f(ab) = f(a)f(b). 2. Peak Finding: Evaluated the Dirichlet polynomial D_F(t; N=10⁶) = Σ_{n=1}^{10⁶} f(n)/n^{1/2+it} over t ∈ [10⁶, 2·10⁶] using 2000 equally spaced grid points. Used Kahan compensated summation for both real and imaginary components to ensure numerical precision. Identified peaks using scipy.signal.find_peaks and extracted the top 20 peaks by magnitude. 3. ω-Class Decomposition: Loaded pre-computed Ω(n) values from omega_values_N1e6.pkl (0-based indexing). For each of the 20 peak locations, computed S_k = Σ_{Ω(n)=k} f(n)/n^{1/2+it} for k=0 to 19 using Kahan summation. Verified that Σ S_k = D_F to numerical precision (< 10⁻¹⁴). 4. Causal Perturbation Analysis: For each peak and each class k ∈ {2, 3, 4}, computed perturbed sum D'_F where S_k → -S_k (equivalent to multiplying by e^{iπ}). Calculated magnitude reduction: Δ_k = (|D_F| - |D'_F|) / |D_F| × 100%. Computed mean and standard deviation across 20 peaks for each k. 5. Statistical Testing: Performed paired t-tests comparing S₃ reductions to S₂ and S₄ reductions (n=20 peaks per comparison). Computed Cohen's d effect sizes for paired samples using d = mean(diff) / std(diff). All computations used double precision floating point arithmetic with Kahan summation to minimize accumulated rounding errors in the complex sums.
