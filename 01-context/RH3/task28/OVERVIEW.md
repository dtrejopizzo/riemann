A rigorous from-scratch implementation of the canonical r metric was developed using Python with NumPy, SciPy, and Pandas. The metric is defined as r(t;N) = Σ_{j≠k, j,k≥1} Re[S_j S̄_k] / Σ_{k≥1} |S_k|², where S_k(t;N) = Σ_{n≤N, ω(n)=k} a_n / n^{1/2+it}. Pre-computation: ω(n) and Ω(n) were computed for all n ≤ 10⁶ using a sieve-based algorithm. S_k computation: For each ω-class k, all n with ω(n)=k were identified, and their contributions a_n/n^{1/2+it} were summed using Kahan compensated summation for maximum numerical precision. r computation: The numerator was computed by direct iteration through all pairs (j,k) with j≠k, calculating Re[S_j S̄_k] for each pair and summing with Kahan summation. The denominator was computed as the direct sum of |S_k|² for all k≥1. S_0 (containing only n=1) was excluded, consistent with the ω-class decomposition framework. Gold standard validation: Computed r for zeta at N=10⁵, t=150000.0, yielding r = -0.823083265909964 (15 decimal places). Mathematical correctness was verified through the Parseval-type identity: |D_F|² = Σ_k |S_k|² + Σ_{j≠k} Re[S_j S̄_k], which held to machine precision (error < 10⁻¹⁴). N-dependence analysis: For N ∈ {10⁴, 10⁵} and functions {zeta, liouville}, |D_F(t;N)| was evaluated over t ∈ [N, 2N] with 1000-2000 uniformly spaced points. Peaks were identified using scipy.signal.find_peaks with prominence=1.0 and distance=5. The top 200 peaks (or all peaks if fewer) were retained. For each peak location, r was computed using the canonical function. Peaks were binned into quartiles by height (Q1=lowest 25%, Q4=highest 25%), and mean r was calculated for each quartile. Trough analysis: To understand the behavior at low |D_F|, troughs (local minima) were identified by inverting the modulus signal and applying peak finding. A sample of 20 troughs was analyzed for r values. Correlation analysis: Pearson correlation coefficients between peak height and r were computed for each function and N value. Implementation validated that the identity |D_F|² = (S_0 + Σ_{k≥1} S_k)(S̄_0 + Σ_{k≥1} S̄_k) holds exactly, confirming mathematical correctness of the ω-class decomposition.
